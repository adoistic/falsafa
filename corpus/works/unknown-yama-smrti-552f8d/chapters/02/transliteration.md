prāyaścittīyatāṃ prāpya ye narās tv akṛtavratāḥ  /

teṣāṃ daṇḍaṃ vratañ cāpi kramaśaś coditaṃ śṛṇu  // YSS_2.1 //

svayam eva tu yaḥ pāpaṃ nivedya vratam ācaret  /

tasmin damo na prayojyo vrataṃ daṇḍaś ca gṛhyate  // YSS_2.2 //

na pātake 'rtthadaṇḍo 'sti tulyadoṣe vidaṇḍanam  /

tatra vratam akurvāṇaṃ sahasraṃ dāpayed damaṃ  // YSS_2.3 //

upapātakayukto yo mānavo nācared vrataṃ  /

sadāpy asāhasaṃ pūrvaṃ prāyaścittaṃ viśodhanaṃ  // YSS_2.4 //

cāṣamaṇḍūkamārjāra- godholūkāṃś ca vāyasān  /

hatvā śatam avāpnoti vipro nakulam eva hi  // YSS_2.5 //

mayūrahaṃsabhāsāñ ca tathā śyenaṃś ca kukkuṭaṃ  /

triṃśat paṇam avāpnoti tathaiva vanabarhiṇaṃ  // YSS_2.6 //

vipram agniparityaktaṃ saṃdhyopāsanavarjitaṃ  /

śūdrapreṣyakarañ caiva rājā rāṣṭrāt pravāsayet  // YSS_2.7 //

utsṛṣṭāgnis tu yo vipras tv asnāto 'mantrato 'pi vā  /

upasthānārcano sūryaṃ sandhyāṃ nopāsate dvijaḥ  // YSS_2.8 //

akartā nityayajñānāṃ bhuñjāno vṛṣalāśanaṃ  /

jīvamāno 'pi vā śastraiḥ pratyahaṃ paṇam āpnuyāt  // YSS_2.9 //

śūdrāsu śuklasektāraṃ vṛṣalāyāṃ ca vāñchakaṃ  /

vipraṃ sarvasvam ādāya lalāṭe bhagam aṅkayet  // YSS_2.10 //

bhagacihnaṃ madyacihnaṃ surācihnaṃ lalāṭake  /

veśyāsu śuklasektāraṃ sarvam ādāya bhūmipaḥ  // YSS_2.11 //

apasnāto divāsnātaḥ parānnam api vātmanaḥ  /

bhoktā naṣṭakriyas tasya rājñaḥ svārtthaṃ pracakṣate  // YSS_2.12 //

viprasyoktaparityaktaṃ vṛṣalāśanasevinaḥ  /

pratyabdaṃ sarvam ādāya vedavidbhyo nivedayet  // YSS_2.13 //

parītaṃ vottaraṃ vāsaḥ adharmadaṇḍasāhasaṃ  /

striyam eva prakurvāṇaṃ dviguṇaṃ dāpayed damaṃ  // YSS_2.14 //

bālāndharogivṛddhānāṃ na tu daṇḍo vidhīyate  /

visṛjya tu naro mūtraṃ rathyādau doṣam āpnuyāt  // YSS_2.15 //

tathā purīṣam uddiṣṭaṃ chardiś ca na bhavet pathi  /

dviguṇaṃ daṇḍam āpnoti mūtraṃ sthitvā samutsṛjet  // YSS_2.16 //

māhendrīṃ vāruṇīñ cāpi diśaṃ vīkṣya tu mūtrayan  /

purīṣaṃś ca tathā muñcan daṇḍaṃ dviguṇam āvahet  // YSS_2.17 //

stenaḥ piśunavaktā ca sāhasī vṛṣalīpatiḥ  /

tyaktāvāritā vijñeyāḥ pañcaite grāmakaṇṭakāḥ  // YSS_2.18 //

visṛṣṭvānuparīkṣyaitān rājā rāṣṭreṣu kaṇṭakān  /

anutsṛṣṭā hi lokasya tv anṛtaṃ kurvate tamaḥ  // YSS_2.19 //

viṣṭādipūrṇasaṃkṣiptaṃ raktamālyavibhūṣitaṃ  /

poṣitaṃ stenakāryeṇa stenaṃ rājā nihanyate  // YSS_2.20 //

śvapadādi mukhe nyasya pariyāpya kharājinaṃ  /

piśunaṃ kharam āropya hṛtavittaṃ vivāsayet  // YSS_2.21 //

bhittvā gṛhāṃs tu hṛtvā svaṃ kṛtvopaskaram eva ca  /

rājā sāhasikaṃ rāṣṭrāc caṇḍāḷotsṛṣṭam utsṛjet  // YSS_2.22 //

vratabhaṅgaṃ surābhāṇḍaṃ dvayaṃ saṃbadhyate tataḥ  /

hṛdayaṃ viṣṭhayā liptaṃ vṛṣalīpatim utsṛjet  // YSS_2.23 //

bhasmanāpi ca liptāṅgaṃ śavakeśāsthidhūpitaṃ  /

tyaktāgniṃ sarvam ādāya rājā rāṣṭrāt pravāsayet  // YSS_2.24 //

śavakeśair vījyamānaṃ tanmālābaddhaśekharaṃ  /

digvāsaṃ gamayed rājā naraṃ samayalaṅghinaṃ  // YSS_2.25 //

śūdrāsūtpāditāpatyaṃ rājasaṃ brāhmaṇādhamaṃ  /

grāmād bahiḥ prakurvīta naraṃ saṃsargapātakaṃ  // YSS_2.26 //

tasyāṃ tu śuklasektāraṃ miśritya saha sādhubhiḥ  /

bhojayet tulyapaṅktau tu prāpnuyāt prathamaṃ damaṃ  // YSS_2.27 //

vittaṃ dattaṃ tathāmārgād adhītañ ca dvijādibhiḥ  /

jāyate viphalaṃ sarvaṃ bhāṣaṇāt piśunair naraiḥ  // YSS_2.28 //

devabrahmasvayonyas tu chidradātā vilaṃbhane  /

dehabhedam avāpnoti vipro hṛtvākhilaṃ dhanaṃ  // YSS_2.29 //

chidrapradātā stenānāṃ stena eva sa ucyate  /

tasya hṛtvā tu sarvasvaṃ śvapadaṃ tu mukhe 'ṅkayet  // YSS_2.30 //

naktaṃ rātrau tu janitā liṅgāj jānāti mānavaḥ  /

tathā vai śūnyavaktāraṃ jñātvā liṅgaiḥ pravāsayet  // YSS_2.31 //

asatyavādinaṃ vāde daṇḍitvā sākṣiṇaṃ nṛpaḥ  /

yaś chadati hi pāpānāṃ viṃśatiṃ vratam ādiśet  // YSS_2.32 //

vratacāryaś caran bhikṣām aniṣṭvā ca hutāśanaṃ  /

bhuñjānaḥ prāpnuyāt grāsaṃ kāle kāle tv anāturaḥ  // YSS_2.33 //

ācāryasya pitur bhrātuḥ śvaśurasya nṛpasya ca  /

pratyutthānaṃ naro 'kurvans trīn paṇān dāpyate damaṃ  // YSS_2.34 //

āsane tu samāsīna uttame 'sminn avasthite  /

śūdraṣ ṣaṭpadam āpnoti gacchatāprasthitena tu  // YSS_2.35 //

unmānañ ca tathāmānaṃ variṣṭhalaghu vāgṛte  /

dṛṣyate yasya tasya svaṃ rājña ity eva kalpayet  // YSS_2.36 //

miśritaṃ tilatailena vikrītaṃ ghṛtakākhyayā  /

vaṇik caturguṇaṃ dāpyaṃ tailaṃ vā miśritaṃ ghṛtaiḥ  // YSS_2.37 //

marīcaṃ khaṇḍapāṣāṇaiḥ lavaṇañ ca tathā mṛdā  /

hiṅguñ ca vṛkṣaniryāsaiḥ tathā dāpyaś caturguṇaṃ  // YSS_2.38 //

dhānyaṃ pulākapāṣāṇaiḥ khalu miśritya vā punaḥ  /

vikretā tatsamaṃ dāpyo dadyāt tasyāpi miśritaṃ  // YSS_2.39 //

tulyair atulyair vā dravyaiḥ miśrīkṛtya tato nṛṇāṃ  /

damaś caturguṇaḥ proktas tena nityaṃ samācaret  // YSS_2.40 //

pretasammārjanaṃ malaṃ kapālaṃ tuṣam eva ca  /

pathi grāmasya vikṣeptā dāpyaḥ ..... paṇaṃ  // YSS_2.41 //

ucchiṣṭasalilasrāvaṃ gṛhī yaḥ parimuñcati  /

dviguṇaṃ daṇḍam āpnoti moktā rūpye ca daṇḍyate  // YSS_2.42 //

hatvā kakaraṃ divā tu tripaṇaṃ brāhmaṇo bhavet  /

śaśaṃ kūrmañ ca godhāñ ca meṣaṃ hatvā ca ṣoḍaśa  // YSS_2.43 //

kṣīraṃ kṣīravikārañ ca lavaṇaṃ tailam eva ca  /

lākṣāñ ca vikrayaṃ kurvan vipraḥ prāpnoti madhyamaṃ  // YSS_2.44 //

tvacañ ca vikrayaṃ kurvan prāpnuyān madhyamaṃ damaṃ  // YSS_2.45 //

janakañ janakādhyakṣo nikarṣan viṣamopame  /

damaṃ paṇam avāpnoti bhoktā ca viṣamopame  // YSS_2.46 //

vipram utpāditāpatyaṃ vṛṣalyāṃ yo vahed ataḥ  /

sahasraṃ damam āpnoti prasūtiḥ prathamaṃ bhavet  // YSS_2.47 //

bhuñjāno yasya pakvānnaṃ vipras tu vṛṣalīpateḥ  /

paṇañ caturdaśāpnoti dviguṇaṃ tasya sūtake  // YSS_2.48 //

śrāddhaṃ tu tadgṛhe bhuktvā paṇan tāvad avāpnuyāt  /

sāhasaṃ pūrvam āpnoti viśeṣāc cānimittataḥ  // YSS_2.49 //

anvaṣṭakām aṣṭakāṃ vā bhuñjāno vṛṣalīpateḥ  /

paṇāñ caturdaśāpnoti tathā bhoktā ca māsikaṃ  // YSS_2.50 //

kramam apy aṣṭakādeśāt gṛhṇāno vṛṣalālayāt  /

ṣaṇṇavatyarddham āpnoti bhuktvā pūrvaṃ tu tadgṛhe  // YSS_2.51 //

madyādīnāṃ tv abhakṣyāṇāṃ vipraḥ prāpnoti ṣoḍaśa  /

dviguṇaṃ pratipīḍānāṃ matsyānāṃ āpnuyāc chataṃ  // YSS_2.52 //

evaṃ dviguṇatāṃ dadyāt doṣeṣv eteṣu somapāḥ  /

kharāśvasūkarādīnaṃ bhakṣaṇe patitaḥ smṛtaḥ  // YSS_2.53 //

dārvasthiśṛṅgapāṣāṇair yas tu bhāryābhṛtiṃ dvijaḥ  /

sahasraṃ damam āpnoti kurvāṇo vapanaṃ yadi  // YSS_2.54 //

aṅgeṣu vyaktadeśeṣu yo jāyāṃ karajair vraṇaṃ  /

kartā kārayitā svāṅgair dāpyaḥ syāt prathamaṃ damaṃ  // YSS_2.55 //

yoṣitmukhaṃ tu bhuñjāno pītvā vāpidadaṃ dvijaḥ  /

tathā bhojayitā svaṃ vā dāpyaḥ syāt pūrvasāhasaṃ  // YSS_2.56 //

atyarthaṃ vedayaṃś caiva brāhmaṇaḥ patito bhavet  /

striyaṃ tathāśvavad gacchamś cuṃbayan guhyam eva vā  // YSS_2.57 //

prayatnāc chuklam utsṛjya vipraḥ pañcāśad āpnuyāt  /

maithunaṃ goṣu gatvā tu tathā dāpyo bhavec chataṃ  // YSS_2.58 //

gacchaṃs tathājayonau tu prāpyate pūrvasāhasaṃ  /
