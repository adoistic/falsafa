**Vishnu Smriti 78**

By constantly performing a *śrāddha* (श्राद्ध) on Sunday, he obtains health. // Vi_78.1 //

Good fortune on Monday. // Vi_78.2 //

Victory in battle on Tuesday. // Vi_78.3 //

All desires on Wednesday. // Vi_78.4 //

The desired knowledge on Thursday. // Vi_78.5 //

Wealth on Friday. // Vi_78.6 //

Long life on Saturday. // Vi_78.7 //

Heaven in the *Kṛttikā* (कृत्तिका) constellation. // Vi_78.8 //

Offspring in the *Rohiṇī* (रोहिणी) constellation. // Vi_78.9 //

Spiritual splendor in the *Saumya* (सौम्य) (Mrigashirsha) constellation. // Vi_78.10 //

Success in his undertakings in the *Raudra* (रौद्र) (Ardra) constellation. // Vi_78.11 //

Land in the *Punarvasu* (पुनर्वसु) constellation. // Vi_78.12 //

Nourishment in the *Puṣya* (पुष्य) constellation. // Vi_78.13 //

Prosperity in the *Sārpa* (सार्प) (Ashlesha) constellation. // Vi_78.14 //

All desires in the *Paitrya* (पैत्र्य) (Magha) constellation. // Vi_78.15 //

Good fortune in the *Bhāgya* (भाग्य) (Purva Phalguni) constellation. // Vi_78.16 //

Wealth in the *Āryamaṇa* (आर्यमण) (Uttara Phalguni) constellation. // Vi_78.17 //

Supremacy among his kinsmen in the *Hasta* (हस्त) constellation. // Vi_78.18 //

Sons of beautiful form in the *Tvāṣṭra* (त्वाष्ट्र) (Chitra) constellation. // Vi_78.19 //

Success in trade in the *Svāti* (स्वाति) constellation. // Vi_78.20 //

Gold in the *Viśākhā* (विशाखा) constellation. // Vi_78.21 //

Friends in the *Maitra* (मैत्र) (Anuradha) constellation. // Vi_78.22 //

Sovereignty in the *Śākra* (शाक्र) (Jyeshtha) constellation. // Vi_78.23 //

Agriculture in the *Mūla* (मूल) constellation. // Vi_78.24 //

Success in sea voyages in the *Āpya* (आप्य) (Purva Ashadha) constellation. // Vi_78.25 //

All desires in the *Vaiśvadeva* (वैश्वदेव) (Uttara Ashadha) constellation. // Vi_78.26 //

Supremacy in the *Abhijit* (अभिजित्) constellation. // Vi_78.27 //

All desires in the *Śravaṇa* (श्रवण) constellation. // Vi_78.28 //

Salt in the *Vāsava* (वासव) (Dhanishtha) constellation. // Vi_78.29 //

Health in the *Vāruṇa* (वारुण) (Shatabhishak) constellation. // Vi_78.30 //

Base metals in the *Āja* (आज) (Purva Bhadrapada) constellation. // Vi_78.31 //

A house in the *Āhirbudhnya* (आहिर्बुध्न्य) (Uttara Bhadrapada) constellation. // Vi_78.32 //

Cows in the *Pauṣṇa* (पौष्ण) (Revati) constellation. // Vi_78.33 //

Horses in the *Āśvina* (आश्विन) (Ashvini) constellation. // Vi_78.34 //

Long life in the *Yāmya* (याम्य) (Bharani) constellation. // Vi_78.35 //

A house and beautiful women on the first day of the lunar fortnight. // Vi_78.36 //

A boon-giving daughter on the second. // Vi_78.37 //

All desires on the third. // Vi_78.38 //

Cattle on the fourth. // Vi_78.39 //

Beautiful sons on the fifth. // Vi_78.40 //

Victory in gambling on the sixth. // Vi_78.41 //

Agriculture on the seventh. // Vi_78.42 //

Trade on the eighth. // Vi_78.43 //

Cattle on the ninth. // Vi_78.44 //

Horses on the tenth. // Vi_78.45 //

Sons endowed with spiritual splendor on the eleventh. // Vi_78.46 //

Gold and silver on the twelfth. // Vi_78.47 //

Good fortune on the thirteenth. // Vi_78.48 //

All desires on the fifteenth. // Vi_78.49 //

The fourteenth day is praised for the *śrāddha* (श्राद्ध) rite for those killed by weapons. // Vi_78.50 //

And on this subject, there are verses sung by the ancestors (*pitṛgītā gāthā* (पितृगीता गाथा)). // Vi_78.51 //

> May some excellent man be born in our family, /
> Who, in the rainy season, in the dark fortnight, on the thirteenth day, with a concentrated mind, || Vi_78.52 ||
>
> Would perform a *śrāddha* (श्राद्ध) with rice-pudding overflowing with honey, /
> And for the entire month of *Kārttika* (कार्तिक), in the early morning shadow of an elephant. || Vi_78.53 ||
