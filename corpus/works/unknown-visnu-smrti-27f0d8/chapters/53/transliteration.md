Vi_53

athāgamyāgamane mahāvratavidhānenābdaṃ cīravāsā vane prājāpatyaṃ kuryāt // Vi_53.1 //

paradāragamane ca // Vi_53.2 //

govrataṃ gogamane ca // Vi_53.3 //

puṃsy ayonāv ākāśe 'psu divā goyāne ca savāsāḥ snānam ācaret // Vi_53.4 //

caṇḍālīgamane tatsāmyam āpnuyāt // Vi_53.5 //

ajñānataś cāndrāyaṇadvayaṃ kuryāt // Vi_53.6 //

paśuveśyāgamane ca prājāpatyam // Vi_53.7 //

sakṛd duṣṭā ca strī yat puruṣasya paradāre tadvrataṃ kuryāt // Vi_53.8 //

yat karoty ekarātreṇa vṛṣalīsevanād dvijaḥ  /

tad bhaikṣyabhug japan nityaṃ tribhir varṣair vyapohati  || Vi_53.9 ||
