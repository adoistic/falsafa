**Vishnu Smriti 85**

Now, a *śrāddha* (श्राद्ध) performed at Puṣkara is imperishable. // Vi_85.1 //

And so are muttered prayers, oblations, and austerities. // Vi_85.2 //

By merely bathing at Puṣkara, one is purified from all sins. // Vi_85.3 //

Likewise at Gayāśīrṣa. // Vi_85.4 //

At the Vaṭa (banyan tree). // Vi_85.5 //

On the Amarakaṇṭaka mountain. // Vi_85.6 //

On the Varāha mountain. // Vi_85.7 //

Anywhere on the bank of the Narmadā. // Vi_85.8 //

On the bank of the Yamunā. // Vi_85.9 //

And especially on the Ganges. // Vi_85.10 //

At Kuśāvarta. // Vi_85.11 //

At Binduka. // Vi_85.12 //

On the Nīla mountain. // Vi_85.13 //

At Kanakhala. // Vi_85.14 //

At Kubjāmra. // Vi_85.15 //

At Bhṛgutuṅga. // Vi_85.16 //

At Kedāra. // Vi_85.17 //

At Mahālaya. // Vi_85.18 //

At Naḍantikā. // Vi_85.19 //

At Sugandhā. // Vi_85.20 //

At Śākambharī. // Vi_85.21 //

At the Phalgutīrtha. // Vi_85.22 //

At the Mahāgaṅgā. // Vi_85.23 //

In the village of Trihalikā. // Vi_85.24 //

At Kumāradhārā. // Vi_85.25 //

At Prabhāsa. // Vi_85.26 //

Anywhere on the Sarasvatī, especially. // Vi_85.27 //

At Gaṅgādvāra. // Vi_85.28 //

And at Prayāga. // Vi_85.29 //

At the confluence of the Ganges and the ocean. // Vi_85.30 //

Constantly in the Naimiṣa forest. // Vi_85.31 //

And especially in Vārāṇasī. // Vi_85.32 //

In the hermitage of Agastya. // Vi_85.33 //

In the hermitage of Kaṇva. // Vi_85.34 //

On the Kauśikī. // Vi_85.35 //

On the bank of the Sarayū. // Vi_85.36 //

At the confluence of the Śoṇa and the Jyotiṣā. // Vi_85.37 //

On the Śrīparvata mountain. // Vi_85.38 //

At Kālodaka. // Vi_85.39 //

At Uttaramānasa. // Vi_85.40 //

At Baḍabā. // Vi_85.41 //

At Mataṅgavāpī. // Vi_85.42 //

At Saptārṣa. // Vi_85.43 //

At Viṣṇupada. // Vi_85.44 //

At Svargamārgapada. // Vi_85.45 //

On the Godāvarī. // Vi_85.46 //

On the Gomatī. // Vi_85.47 //

On the Vetravatī. // Vi_85.48 //

On the Vipāśā. // Vi_85.49 //

On the Vitastā. // Vi_85.50 //

On the bank of the Śatadrū. // Vi_85.51 //

On the Candrabhāgā. // Vi_85.52 //

On the Irāvatī. // Vi_85.53 //

On the bank of the Sindhu. // Vi_85.54 //

In the southern Pañcanada. // Vi_85.55 //

At Ausaja. // Vi_85.56 //

And in these and other sacred bathing places. // Vi_85.57 //

In the best of rivers. // Vi_85.58 //

In all natural places. // Vi_85.59 //

On riverbanks. // Vi_85.60 //

At springs. // Vi_85.61 //

On mountains. // Vi_85.62 //

In groves. // Vi_85.63 //

In forests. // Vi_85.64 //

In gardens. // Vi_85.65 //

In houses smeared with cow-dung. // Vi_85.66 //

In pleasant places. // Vi_85.67 //

And on this subject, there are verses sung by the ancestors (*pitṛgītā gāthā* (पितृगीता गाथा)). // Vi_85.68 //

> May that being be in our family who will give us handfuls of water, /
> In rivers with abundant water, especially in cool ones. || Vi_85.69 ||
>
> May some excellent man be born in our family, /
> Who would perform a *śrāddha* (श्राद्ध) for us with a concentrated mind at the Vaṭa tree in Gayāśīrṣa. || Vi_85.70 ||
>
> Many sons are to be desired, so that at least one may go to Gayā, /
> Or perform a horse-sacrifice, or set free a dark-blue bull. || Vi_85.71 ||
