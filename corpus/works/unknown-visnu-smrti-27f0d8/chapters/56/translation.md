**Vishnu Smriti 56**

Now, therefore, there are the purifiers of all the Vedas. // Vi_56.1 //

By muttering and offering oblations with which, the twice-born are purified from their sins. // Vi_56.2 //

The Aghamarṣaṇa. // Vi_56.3 //

The Devakṛta. // Vi_56.4 //

The Śuddhavatīs. // Vi_56.5 //

The Taratsamandīya. // Vi_56.6 //

The Kūśmāṇḍīs. // Vi_56.7 //

The Pāvamānīs. // Vi_56.8 //

The Durgāsāvitrī. // Vi_56.9 //

The Atīṣaṅgās. // Vi_56.10 //

The Padastomas. // Vi_56.11 //

The Sāmans and the Vyāhṛtis. // Vi_56.12 //

The Bhāruṇḍas. // Vi_56.13 //

The Candrasāma. // Vi_56.14 //

The two Sāmans of the Puruṣavrata. // Vi_56.15 //

The Abliṅga. // Vi_56.16 //

The Bārhaspatya. // Vi_56.17 //

The Gosūkta (Cow-hymn). // Vi_56.18 //

The Aśvasūkta (Horse-hymn). // Vi_56.19 //

The two Sāmans and the Candrasūktas. // Vi_56.20 //

The Śatarudriya. // Vi_56.21 //

The Atharvaśiras. // Vi_56.22 //

The Trisuparṇa. // Vi_56.23 //

The Mahāvrata. // Vi_56.24 //

The Nārāyaṇīya. // Vi_56.25 //

And the Puruṣasūkta. // Vi_56.26 //

> The three Ājyadohas and the Rathantara, the Agnivrata, the Vāmadevya, and the Bṛhat, /
> These, when sung, purify living beings; one obtains the memory of past lives, if one wishes. || Vi_56.27 ||
