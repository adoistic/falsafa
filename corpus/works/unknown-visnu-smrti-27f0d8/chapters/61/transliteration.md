Vi_61

atha pālāśaṃ dantadhāvanaṃ nādyāt // Vi_61.1 //

naiva śleṣmātakāriṣṭavibhītakadhavadhanvanajam // Vi_61.2 //

na ca bandhūkanirguṇḍīśigrutilvatindukajam // Vi_61.3 //

na ca kovidāraśamīpīlupippaleṅgudaguggulujam // Vi_61.4 //

na pāribhadrakāmlikāmocakaśālmalīśaṇajam // Vi_61.5 //

na madhuram // Vi_61.6 //

nāmlam // Vi_61.7 //

nordhvaśuṣkam // Vi_61.8 //

na suṣiram // Vi_61.9 //

na pūtigandhi // Vi_61.10 //

na picchilam // Vi_61.11 //

na dakṣiṇāparābhimukhaḥ // Vi_61.12 //

adyāc codaṅmukhaḥ prāṅmukho vā // Vi_61.13 //

vaṭāsanārkakhadirakarañjabadarasarjanimbārimedāpāmārgamālatīkakubhabilvānām anyatamam // Vi_61.14 //

kaṣāyaṃ tiktaṃ kaṭukaṃ ca // Vi_61.15 //

kanīnyagrasamasthaulyaṃ sakūrcaṃ dvādaśāṅgulam  /

prātarbhūtvā ca yatavāg bhakṣayed dantadhāvanam  || Vi_61.16 ||

prakṣālya bhaṅktvā taj jahyāc chucau deśe prayatnataḥ  /

amāvāsyāṃ na cāśnīyād dantakāṣṭhaṃ kadācana  || Vi_61.17 ||
