**Vishnu Smriti 69**

He shall not approach his wife on the eighth, fourteenth, and fifteenth days of the lunar fortnight. // Vi_69.1 //

Not after having eaten at a *śrāddha*. // Vi_69.2 //

Not after having given at one. // Vi_69.3 //

Not when he has been invited to a *śrāddha*. // Vi_69.4 //

Not when he is under a vow. // Vi_69.5 //

Not when he has been initiated for a sacrifice. // Vi_69.6 //

Not in a temple, a cremation ground, or a deserted house. // Vi_69.7 //

Not at the roots of trees. // Vi_69.8 //

Not during the day. // Vi_69.9 //

Not at the twilights. // Vi_69.10 //

Not a woman who is unclean. // Vi_69.11 //

Not when he is unclean. // Vi_69.12 //

Not a woman who is unanointed. // Vi_69.13 //

Not when he is unanointed. // Vi_69.14 //

Not a woman who is sick. // Vi_69.15 //

Not when he is sick. // Vi_69.16 //

> Not one with a deficient limb, not one with a superfluous limb, and likewise not one who is older; /
> One who wishes to live a long life shall not approach a pregnant woman. || Vi_69.17 ||
