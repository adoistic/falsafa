Vi_66

na naktaṃ gṛhītenodakena devapitṛkarma kuryāt // Vi_66.1 //

cadanamṛgamadadārukarpūrakuṅkumajātīphalavarjam anulepanaṃ na dadyāt // Vi_66.2 //

na vāso nīlīraktam // Vi_66.3 //

na maṇisuvarṇayoḥ pratirūpam alaṃkaraṇam // Vi_66.4 //

nograghandhi // Vi_66.5 //

nāgandhi // Vi_66.6 //

na kaṇṭakijam // Vi_66.7 //

kaṇṭakijam api śuklaṃ sugandhikaṃ tu dadyāt // Vi_66.8 //

raktam api kuṅkumaṃ jalajaṃ ca dadyāt // Vi_66.9 //

na dhūpārthe jīvajātam // Vi_66.10 //

na ghṛtatailaṃ vinā kiṃcana dīpārthe // Vi_66.11 //

nābhakṣyaṃ naivedyārthe // Vi_66.12 //

na bhakṣye apy ajāmahiṣīkṣīre // Vi_66.13 //

pañcanakhamatsyavarāhamāṃsāni ca // Vi_66.14 //

prayataś ca śucir bhūtvā sarvam eva nivedayet  /

tanmanāḥ sumanā bhūtvā tvarākrodhavivarjitaḥ  || Vi_66.16 ||
