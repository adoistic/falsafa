Vi_3

atha rājadharmāḥ // Vi_3.1 //

prajāparipālanam // Vi_3.2 //

varṇāśramāṇāṃ sve sve dharme vyavasthāpanam // Vi_3.3 //

rājā ca jāṅgalaṃ paśavyaṃ sasyopetaṃ deśam āśrayet // Vi_3.4 //

vaiśyaśūdraprāyaṃ ca // Vi_3.5 //

tatra dhanvanṛmahīvārivṛkṣagiridurgāṇām anyatamaṃ durgam āśrayet // Vi_3.6 //

tatrasthaś ca svasvagrāmādhipān kuryāt // Vi_3.7 //

daśādhyakṣān // Vi_3.8 //

śatādhyakṣān // Vi_3.9 //

deśādhyakṣāṃś ca // Vi_3.10 //

grāmadoṣāṇāṃ grāmādhyakṣaḥ parihāraṃ kuryāt // Vi_3.11 //

aśakto daśagrāmādhyakṣāya nivedayet // Vi_3.12 //

so 'py aśaktaḥ śatādhyakṣāya // Vi_3.13 //

so 'py aśakto deśādhyakṣāya // Vi_3.14 //

deśādhyakṣo 'pi sarvātmanā doṣam ucchindyāt // Vi_3.15 //

ākaraśulkataranāgavaneśv āptān niyuñjīta // Vi_3.16 //

dharmiṣṭhān dharmakāryeṣu // Vi_3.17 //

nipuṇān arthakāryeṣu // Vi_3.18 //

śūrān saṃgrāmakarmasu // Vi_3.19 //

ugrān ugreṣu // Vi_3.20 //

ṣaṇḍhān strīṣu // Vi_3.21 //

prajābhyo balyarthaṃ saṃvatsareṇa dhānyataḥ ṣaṣṭham aṃśam ādadyāt // Vi_3.22 //

sarvasasyebhyaś ca // Vi_3.23 //

dvikaṃ śataṃ paśuhiraṇyebhyo vastrebhyaś ca // Vi_3.24 //

māṃsamadhughṛtauṣadhigandhapuṣpamūlaphalarasadārupatrājinamṛdbhāṇḍāśmabhāṇḍavaidalebhyaḥ ṣāṣṭhabhāgaṃ rājā // Vi_3.25 //

brāhmaṇebhyaḥ karādānaṃ na kuryāt // Vi_3.26 //

te hi rājño dharmakarāḥ // Vi_3.27 //

rājā ca prajābhyaḥ sukṛtaduṣkṛtebhyaḥ ṣaṣṭhāṃśabhāk // Vi_3.28 //

svadeśapaṇyāc ca śulkāṃśaṃ daśamam ādadyāt // Vi_3.29 //

paradeśapaṇyāc ca viṃśatitamam // Vi_3.30 //

śulkasthānād apākrāman sarvāpahāram āpnuyāt // Vi_3.31 //

śilpinaḥ karmajīvinaś ca māsenaikaṃ rājñaḥ karma kuryuḥ // Vi_3.32 //

svāmyamātyadurgakośadaṇḍarāṣṭramitrāṇi prakṛtayaḥ // Vi_3.33 //

taddūṣakāṃś ca hanyāt // Vi_3.34 //

svarāṣṭrapararāṣṭrayoś ca cāracakṣuḥ syāt // Vi_3.35 //

sādhūnāṃ pūjanaṃ kuryāt // Vi_3.36 //

duṣṭāṃś ca hanyāt // Vi_3.37 //

śatrumitrodāsīnamadhyameṣu sāmabhedadānadaṇḍān yathārhaṃ  yathākālaṃ prayuñjīta // Vi_3.38 //

saṃdhivigrahayānāsanasaṃśrayadvaidhībhāvāṃś ca yathākālam āśrayet // Vi_3.39 //

caitre mārgaśīrṣe vā yātrāṃ yāyāt // Vi_3.40 //

parasya vyasane vā // Vi_3.41 //

paradeśāvāptau taddeśadharmān nocchindyāt // Vi_3.42 //

pareṇābhiyuktaś ca sarvātmanā svarāṣṭraṃ gopāyet // Vi_3.43 //

nāsti rājñāṃ samare tanutyāgasadṛśo dharmaḥ // Vi_3.44 //

gobrāhmaṇanṛpamitradhanadārajīvitarakṣaṇāt ye hatās te svargalokabhājaḥ // Vi_3.45 //

varṇasaṃkararakṣaṇārthaṃ ca // Vi_3.46 //

rājā parapurāvāptau tatra tatkulīnam abhiṣiñcet // Vi_3.47 //

na rājakulam ucchindyāt // Vi_3.48 //

anyatrākulīnarājakulāt // Vi_3.49 //

mṛgayākṣastrīpānābhiratiṃ pariharet // Vi_3.50 //

vākpāruṣyadaṇḍapāruṣye ca // Vi_3.51 //

nārthadūṣaṇaṃ kuryāt // Vi_3.52 //

ādyadvārāṇi nocchindyāt // Vi_3.53 //

nāpātravarṣī syāt // Vi_3.54 //

ākarebhyaḥ sarvam ādadyāt // Vi_3.55 //

nidhiṃ labdhvā tadardhaṃ brāhmaṇebhyo dadyāt // Vi_3.56 //

dvitīym ardhaṃ kośe praveśayet // Vi_3.57 //

nidhiṃ brāhmaṇo labdhvā sarvam ādadyāt // Vi_3.58 //

kṣatriyaś caturtham aṃśaṃ rājñe dadyāt, caturtham aṃśaṃ brāhmaṇebhyaḥ, ardham ādadyāt // Vi_3.59 //

vaiṣyas tu caturtham aṃśaṃ rājñe dadyāt, brāhmaṇebhyo 'rdhaṃ, caturtham aṃśam ādadyāt // Vi_3.60 //

śūdraś cāvāptaṃ dvādaśadhā vibhajya pañcāṃśān rājñe dadyāt, pañcāṃśān brāhmaṇebhyaḥ, aṃśadvayam ādadyāt // Vi_3.61 //

aniveditavijñātasya sarvam apaharet // Vi_3.62 //

svanihitād rājñe brāhmaṇavarjaṃ dvādaśam aṃśaṃ dadyuḥ // Vi_3.63 //

paranihitaṃ svanihitam iti bruvaṃs tatsamaṃ daṇḍam āvahet // Vi_3.64 //

bālānāthastrīdhanāni rājā paripālayet // Vi_3.65 //

caurahṛtaṃ dhanam avāpya sarvam eva sarvavarṇebhyo dadyāt // Vi_3.66 //

anavāpya ca svakośād eva dadyāt // Vi_3.67 //

śāntisvastyayanopāyair daivopaghātān praśamayet // Vi_3.68 //

paracakropaghātāṃś ca śastranityatayā // Vi_3.69 //

vedetihāsadharmaśāstrārthakuśalaṃ kulīnam avyaṅgaṃ tapasvinaṃ purohitaṃ ca varayet // Vi_3.70 //

śucīn alubdhān avahitān śaktisaṃpannān sarvārtheṣu ca sahāyān // Vi_3.71 //

svayam eva vyavahārān paśyed vidvadbhir brāhmaṇaiḥ sārdham // Vi_3.72 //

vyavahāradarśane brāhmaṇaṃ vā niyuñjyāt // Vi_3.73 //

janmakarmavratopetāś ca rājñā sabhāsadaḥ kāryāḥ, ripau mitre ca ye samāḥ, kāmakrodhabhayalobhādibhiḥ kāryārthabhir anāhāryāḥ // Vi_3.74 //

rājā ca sarvakāryeṣu sāṃvatsarādhīnaḥ syāt // Vi_3.75 //

devabrāhmaṇān satatam eva pūjayet // Vi_3.76 //

vṛddhasevī bhavet // Vi_3.77 //

yajñayājī ca // Vi_3.78 //

na cāsya viṣaye brāhmaṇaḥ kṣudhārto 'vasīdet // Vi_3.79 //

na cānyo 'pi satkarmanirataḥ // Vi_3.80 //

brāhmaṇebhyaś ca bhuvaṃ pratipādayet // Vi_3.81 //

yeṣāṃ ca pratipādayet teṣāṃ svavaṃśyān bhuvaḥ parimāṇaṃ dānacchedopavarṇanaṃ ca paṭe tāmrapaṭṭe vā likhitaṃ svamudrāṅkitaṃ cāgāminṛpativijñāpanārthaṃ dadyāt // Vi_3.82 //

paradattāṃ ca bhuvaṃ nāpaharet // Vi_3.83 //

brāhmaṇebhyaḥ sarvadāyān prayacchet // Vi_3.84 //

sarvatas tv ātmānaṃ gopāyet // Vi_3.85 //

sudarśanaś ca syāt // Vi_3.86 //

viṣaghnāgadamantradhārī ca // Vi_3.87 //

nāparīkṣitam upayuñjyāt // Vi_3.88 //

smitapūrvābhibhāṣī syāt // Vi_3.89 //

vadhyeṣv api na bhruṃkuṭīm ācaret // Vi_3.90 //

aparādhānurūpaṃ ca daṇḍaṃ daṇḍyeṣu dāpayet // Vi_3.91 //

samyagdaṇḍapraṇayanaṃ kuryāt // Vi_3.92 //

dvitīyam aparādhaṃ na sa kasyacit kṣameta // Vi_3.93 //

svadharmam apālayan nādaṇḍyo nāmāsti rājñām // Vi_3.94 //

yatra śyāmo lohitākṣo daṇḍaś carati nirbhayaḥ  /

prajās tatra vivardhante netā cet sādhu paśyati  || Vi_3.95 ||

svarāṣṭro nyāyadaṇḍaḥ syād bhṛśadaṇḍaś ca śatruṣu  /

suhṛtsv ajihmaḥ snigdheṣu brāhmaṇeṣu kṣamānvitaḥ  || Vi_3.96 ||

evaṃvṛttasya nṛpateḥ śiloñchenāpi jīvataḥ  /

vistīryate yaśo loke tailabindur ivāmbhasi  || Vi_3.97 ||

prajāsukhe sukhī rājā tadduḥkhe yaś ca duḥkhitaḥ  /

sa kīrtiyukto loke 'smin pretya svarge mahīyate  || Vi_3.98 ||
