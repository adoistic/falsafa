Vi_29

yas tūpanīya vratādeśaṃ kṛtvā vedam adhyāpayet tam ācāryaṃ vidyāt // Vi_29.1 //

yas tv enaṃ mūlyenādhyāpayet tam upādhyāyam ekadeśaṃ vā // Vi_29.2 //

yo yasya yajñnakarmāṇi kuryāt tam ṛtvijaṃ vidyāt // Vi_29.3 //

nāparīkṣitaṃ yojayet // Vi_29.4 //

nādhyāpayet // Vi_29.5 //

nopanayet // Vi_29.6 //

adharmeṇa ca yaḥ prāha yaś cādharmeṇa pṛcchati  /

tayor anyataraḥ praiti vidveṣaṃ vādhigacchati  || Vi_29.7 ||

dharmārthau yatra na syātāṃ śuśrūṣā vāpi tadvidhā  /

tatra vidyā na vaktavyā śubhaṃ bījam ivoṣare  || Vi_29.8 ||

vidyā ha vai brahmaṇam ājagāma gopāya mā śevadhiṣṭe 'ham asti  /

asūyakāyānṛjave 'yatāya na māṃ brūyā avīryavatī tathā syām  || Vi_29.9 ||

yam eva vidyāḥ śucim apramattaṃ medhāvinaṃ brahmacaryopapannam  /

yas te na druhyet katamac ca nāha tasmai māṃ brūyā nidhipāya brahman  || Vi_29.10 ||
