Vi_47

atha cāndrāyaṇam // Vi_47.1 //

grāsān avikārān aśnīyāt // Vi_47.2 //

tāṃś candrakalābhivṛddhau vardhayet, hānau hrāsayet, amāvāsyāyāṃ nāśnīyāt / eṣa cāndrāyaṇo yavamadhyaḥ // Vi_47.3 //

pipīlikāmadhyo vā // Vi_47.4 //

yasyāmāvāsyā madhye bhavati sa pipīlikāmadhyaḥ // Vi_47.5 //

yasya paurṇamāsī sa yavamadhyaḥ // Vi_47.6 //

aṣṭau grāsān pratidivasaṃ māsam aśnīyāt sa yaticāndrāyaṇaḥ // Vi_47.7 //

sāyaṃ prātaś caturaś caturaḥ sa śiśucāndrāyaṇaḥ // Vi_47.8 //

yathā kathaṃcit ṣaṣṭyonāṃ triśatīṃ māsenāśnīyāt sa sāmānyacāndrāyaṇaḥ // Vi_47.9 //

vratam etat purā bhūmi kṛtvā saptarṣayo 'malāḥ  /

prāptavantaḥ paraṃ sthānaṃ brahmā rudras tathaiva ca  || Vi_47.10 ||
