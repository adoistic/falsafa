Vi_87

atha vaiśākhyāṃ paurṇamāsyāṃ kṛṣṇājinaṃ sakhuraṃ saśṛṅgaṃ suvarṇaśṛṅgaṃ raupyakhuraṃ mauktikalāṅgūlabhūṣitaṃ kṛtvāvike ca vastre prasārayet // Vi_87.1 //

tatas tilaiḥ pracchādayet // Vi_87.2 //

suvarṇanābhiṃ ca kuryāt // Vi_87.3 //

ahatena vāsoyugena pracchādayet // Vi_87.4 //

sarvagandharatnaiś cālaṃkṛtaṃ kuryāt // Vi_87.5 //

catasṛṣu dikṣu catvāri taijasāni pātrāṇi kṣīradadhimadhughṛtapūrṇāni nidhyāyāhitāgnaye brāhmaṇāyālaṃkṛtāya vāsoyugena pracchāditāya dadyāt // Vi_87.6 //

atra ca gāthā bhavanti // Vi_87.7 //

yas tu kṛṣṇājinaṃ dadyāt sakhuraṃ śṛṅgasaṃyutam  /

tilaiḥ pracchādya vāsobhiḥ sarvaratnair alaṃkṛtam  || Vi_87.8 ||

sasamudraguhā tena saśailavanakānanā  /

caturantā bhaved dattā pṛthivī nātra saṃśayaḥ  || Vi_87.9 ||

kṛṣṇājine tilān kṛtvā hiraṇyaṃ madhusarpiṣī  /

dadāti yas tu viprāya sarvaṃ tarati duṣkṛtam  || Vi_87.10 ||
