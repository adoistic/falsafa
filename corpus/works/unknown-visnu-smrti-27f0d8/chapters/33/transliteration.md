Vi_33

atha puruṣasya kāmakrodhalobhākhyaṃ riputrayaṃ sughoraṃ bhavati // Vi_33.1 //

parigrahaprasaṅgād viśeṣeṇa gṛhāśramiṇaḥ // Vi_33.2 //

tenāyam ākrānto 'tipātakamahāpātakānupātakopapātakeṣu pravartate // Vi_33.3 //

jātibhraṃśakareṣu saṃkarīkaraṇeṣV apātrīkaraṇeṣu // Vi_33.4 //

malāvaheṣu prakīrṇakeṣu ca // Vi_33.5 //

trividhaṃ narakasyedaṃ dvāraṃ nāśanam ātmanaḥ  /

kāmaḥ krodhas tathā lobhas tasmād etat trayaṃ jyayet  || Vi_33.6 ||
