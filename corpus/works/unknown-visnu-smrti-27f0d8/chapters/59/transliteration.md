Vi_59

gṛhāśramī vaivāhikāgnau pākayajñān kuryāt // Vi_59.1 //

sāyaṃ prātaś cāgnihotraṃ // Vi_59.2 //

*devatābhyo juhuyāt [devātābhyo] // Vi_59.3 //

candrārkasaṃnikarṣaviprakarṣayor darśapūrṇamāsābhyāṃ yajeta // Vi_59.4 //

pratyayanaṃ paśunā // Vi_59.5 //

śaradgrīṣmayoś ca āgrayaṇena // Vi_59.6 //

vrīhiyavayor vā pāke // Vi_59.7 //

traivārṣikābhyadhikān naḥ // Vi_59.8 //

pratyabdaṃ somena // Vi_59.9 //

vittābhāve iṣṭyā vaiśvānaryā // Vi_59.10 //

yajñārthaṃ bhikṣitam avāptam arthaṃ sakalam eva vitaret // Vi_59.11 //

sāyaṃ prātar vaiśvadevaṃ juhuyāt // Vi_59.12 //

bhikṣāṃ ca bhikṣave dadyāt // Vi_59.13 //

arcitabhikṣādānena godānaphalam āpnoti // Vi_59.14 //

bhikṣvabhāve grāsamātraṃ gavāṃ dadyāt // Vi_59.15 //

vahnau vā prakṣipet // Vi_59.16 //

bhukte 'py anne vidyamāne na bhikṣukaṃ pratyācakṣīta // Vi_59.17 //

kaṇḍanī peṣaṇī cullī udakumbha upaskara iti pañca sūnā gṛhasthasya // Vi_59.18 //

tanniṣkṛtyarthaṃ ca brahmadevabhūtapitṛnarayajñān kuryāt // Vi_59.19 //

svādhyāyo brahmayajñaḥ // Vi_59.20 //

homo daivaḥ // Vi_59.21 //

pitṛtarpaṇaṃ pitryaḥ // Vi_59.22 //

balir bhautaḥ // Vi_59.23 //

nṛyajñaś cātithipūjanam // Vi_59.24 //

devatātithibhṛtyānāṃ pitṝṇām ātmanaś ca yaḥ  /

na nirvapati pañcānām ucchvasan na sa jīvati  || Vi_59.25 ||

brahmacārī yatir bhikṣur jīvanty ete gṛhāśramāt  /

tasmād abhyāgatān etān gṛhastho nāvamānayet  || Vi_59.26 ||

gṛhastha eva yajate gṛhasthas tapyate tapaḥ  /

pradadāti gṛhasthaś ca tasmāc chreṣṭho gṛhāśramī  || Vi_59.27 ||

ṛṣayaḥ pitaro devā bhūtāny atithayas tathā  /

āśāsate kuḍumbibhyas tasmāc chreṣṭho gṛhāśramī  || Vi_59.28 ||

trivargasevāṃ satatānnadānaṃ surārcanaṃ brāhmaṇapūjanaṃ ca  /

svādhyāyasevāṃ pitṛtarpaṇaṃ ca kṛtvā gṛhī śakrapadaṃ prayāti  || Vi_59.29 ||
