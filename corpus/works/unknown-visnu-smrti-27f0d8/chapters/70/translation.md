**Vishnu Smriti 70**

He shall not sleep with wet feet. // Vi_70.1 //

Not with his head to the north or west. // Vi_70.2 //

Not naked. // Vi_70.3 //

Not along the main beam of the house. // Vi_70.4 //

Not in the open air. // Vi_70.5 //

Not on a bed of Palāśa wood. // Vi_70.6 //

Not on one made of five kinds of wood. // Vi_70.7 //

Not on one made from a tree broken by an elephant. // Vi_70.8 //

Not on one made from a tree struck by lightning. // Vi_70.9 //

Not on a broken one. // Vi_70.10 //

Not on the back of a fire. // Vi_70.11 //

Not on one made from a tree watered from a pitcher. // Vi_70.12 //

Not in a cremation ground, a deserted house, or a temple. // Vi_70.13 //

Not in the midst of a restless crowd. // Vi_70.14 //

Not in the midst of women. // Vi_70.15 //

Not on top of grain, cows, a preceptor, a sacred fire, or deities. // Vi_70.16 //

> He shall not sleep while in a state of impurity, nor during the day, nor at the twilights, nor on ashes, /
> Nor in an unclean place, nor in a wet place, nor on the top of a mountain. || Vi_70.17 ||
