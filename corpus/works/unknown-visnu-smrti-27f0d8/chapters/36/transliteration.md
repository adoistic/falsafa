Vi_36

yāgasthasya kṣatriyasya vaiśyasya ca rajasvalāyāś cāntarvatnyāś cātrigotrāyāś cāvijñātasya garbhasya śaraṇāgatasya ca ghātanaṃ brahmahatyāsamānīti // Vi_36.1 //

kauṭasākṣyaṃ suhṛdvadha ity etau surāpānasamau // Vi_36.2 //

brāhmaṇasya bhūmyapaharaṇaṃ nikṣepāpaharaṇaṃ suvarṇasteyasamam // Vi_36.3 //

pitṛvyamātāmahamātulaśvaśuranṛpapatnyabhigamanaṃ gurudāragamanasamam // Vi_36.4 //

pitṛṣvasṛmātṛṣvasṛsvasṛgamanam ca // Vi_36.5 //

śrotriyartvigupādhyāyamitrapatnyabhigamanaṃ ca // Vi_36.6 //

svasuḥ sakhyāḥ sagotrāya uttamavarṇāyāḥ kumāryā antyajāyā rajasvalāyāḥ śaraṇāgatāyāḥ *pravrajitāyā nikṣiptāyāś ca [pravrajitāyāḥ ni] // Vi_36.7 //

anupātakinas tv ete mahāpātakino yathā  /

aśvamedhena śudhyanti tīrthānusaraṇena vā  || Vi_36.8 ||
