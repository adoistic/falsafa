Vi_63

atha yogakṣemārtham īśvaram abhigacchet // Vi_63.1 //

naiko 'dhvānaṃ prapadyeta // Vi_63.2 //

nādhārmikaiḥ sārdham // Vi_63.3 //

na vṛṣalaiḥ // Vi_63.4 //

na dviṣadbhiḥ // Vi_63.5 //

nātipratyūṣasi // Vi_63.6 //

nātisāyam // Vi_63.7 //

na saṃdhyayoḥ // Vi_63.8 //

na madhyāhne // Vi_63.9 //

na saṃnihitapānīyam // Vi_63.10 //

nātitūrṇam // Vi_63.11 //

na satataṃ bālavyādhitārtair vāhanaiḥ // Vi_63.12 //

na hīnāṅgaiḥ // Vi_63.13 //

na dīnaiḥ // Vi_63.14 //

na gobhiḥ // Vi_63.15 //

nādāntaiḥ // Vi_63.16 //

yavasodake vāhanānām adattvā ātmanaḥ kṣuttṛṣṇāpanodanaṃ na kuryāt // Vi_63.17 //

na catuṣpatham adhitiṣṭhet // Vi_63.18 //

na rātrau vṛkṣamūle // Vi_63.19 //

na śūnyālayam // Vi_63.20 //

na tṛṇam // Vi_63.21 //

na paśūnāṃ bandhanāgāram // Vi_63.22 //

na keśatuṣakapālāsthibhasmāṅgārān // Vi_63.23 //

na kārpāsāsthi // Vi_63.24 //

catuṣpathaṃ prakakṣiṇīkuryāt // Vi_63.25 //

devatārcāṃ ca // Vi_63.26 //

prajñātāṃś ca vanasptīn // Vi_63.27 //

agnibrāhmaṇagaṇikāpūrṇakumbhādarśacchatradhvajapatākāśrīvṛkṣavardhamānanandyāvartāṃś ca // Vi_63.28 //

tālavṛntacāmarāśvagajājagodadhikṣīramadhusiddhārthakāṃś ca // Vi_63.29 //

vīṇācandanāyudhārdragomayaphalapuṣpārdraśākagorocanādūrvāprarohāṃś ca // Vi_63.30 //

uṣṇīṣālaṃkāramaṇikanakarajatavastrāsanayānāmiṣāṃś ca // Vi_63.31 //

bhṛṅgāroddhṛtorvarābaddhaikapaśukumārīmīnāṃś ca dṛṣṭvā prayāyād iti // Vi_63.32 //

atha mattonmattavyaṅgān dṛṣṭvā nivarteta // Vi_63.33 //

vāntaviriktamuṇḍajaṭilavāmanāṃś ca // Vi_63.34 //

kāṣāyipravrajitamalināṃś ca // Vi_63.35 //

tailaguḍaśuṣkagomayendhanatṛṇapalāśabhasmāṅgārāṃś ca // Vi_63.36 //

lavaṇaklībāsavanapuṃsakakārpāsarajjunigaḍamuktakeśāṃś ca // Vi_63.37 //

vīnācandanārdraśākoṣṇīṣālaṃkaraṇakumārīs tu prasthānakāle abhinandayed iti // Vi_63.38 //

devabrāhmaṇagurubabhrudīkṣitānāṃ chāyāṃ nākrāmet // Vi_63.39 //

niṣṭhyūtavāntarudhiraviṇmūtrasnānodakāni ca // Vi_63.40 //

na vatsatantrīṃ laṅghayet // Vi_63.41 //

pravarṣati na dhāvet // Vi_63.42 //

na vṛthā nadīṃ taret // Vi_63.43 //

na devatābhyaḥ pitṛbhyaś codakam apradāya // Vi_63.44 //

na bāhubhyām // Vi_63.45 //

na bhinnayā nāvā // Vi_63.46 //

na kūlam adhitiṣṭhet // Vi_63.47 //

na kūpam avalokayet // Vi_63.48 //

na laṅghayet // Vi_63.49 //

vṛddhabhārinṛpasnāta strīrogivaracakriṇām  /

panthā deyā nṛpas tv eṣāṃ mānyaḥ snātaś ca bhūpateḥ  || Vi_63.50 ||
