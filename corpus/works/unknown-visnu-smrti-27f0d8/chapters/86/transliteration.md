Vi_86

atha vṛṣotsargaḥ // Vi_86.1 //

kārttikyām āśvayujyāṃ vā // Vi_86.2 //

tatrādāv eva vṛṣabhaṃ parīkṣeta // Vi_86.3 //

jīvadvatsāyāḥ payasvinyāḥ putram // Vi_86.4 //

sarvalakṣaṇopetam // Vi_86.5 //

nīlam // Vi_86.6 //

lohitaṃ vā mukhapucchapādaśṛṅgaśuklam // Vi_86.7 //

yūthasyācchādakam // Vi_86.8 //

tato gavāṃ madhye susamiddham agniṃ paristīrya pauṣṇaṃ caruṃ payasā śrapayitvā pūṣā gā anvetu na iha ratir iti ca hutvā vṛṣam ayaskāras tv aṅkayet // Vi_86.9 //

ekasmin pārśve cakreṇāparasmin pārśve śūlena // Vi_86.10 //

aṅkitaṃ ca hiraṇyavarṇeti catasṛbhiḥ śaṃ no devīr iti ca snāpayet // Vi_86.11 //

snātam alaṃkṛtaṃ snātālaṃkṛtābhiś catasṛbhir vatsatarībhiḥ sārdham ānīya rudrān puruṣasūktaṃ kūśmāṇḍīś ca japet // Vi_86.12 //

pitā vatsānām iti vṛṣabhasya dakṣiṇe karṇe paṭhet // Vi_86.13 //

imaṃ ca // Vi_86.14 //

vṛṣo hi bhagavān dharmaś catuṣpādaḥ prakīrtitaḥ  /

vṛṇomi tam ahaṃ bhaktyā sa me rakṣatu sarvataḥ  || Vi_86.15 ||

etaṃ yuvānaṃ patiṃ vo dadāmy anena krīḍantīś carata priyeṇa  /

mā hāsmahi prajayā mā tanūbhir mā radhāma dviṣate soma rājan  || Vi_86.16 ||

vṛṣaṃ vatsatarīyuktam aiśānyāṃ kārayed diśi  /

hotur vastrayugaṃ dadyāt suvarṇaṃ kāṃsyam eva ca  || Vi_86.17 ||

ayaskārasya dātavyaṃ vetanaṃ manasepsitam  /

bhojanaṃ bahusarpiṣkaṃ brāhmaṇāṃś cātra bhojayet  || Vi_86.18 ||

utsṛṣṭo vṛṣabho yasmin pibaty atha jalāśaye  /

jalāśayaṃ tat sakalaṃ pitṝṃs tasyopatisṭhati  || Vi_86.19 ||

śṛṅgeṇollikhate bhūmiṃ yatra kvacana darpitaḥ  /

pitṝṇām annapānaṃ tat prabhūtam upatiṣṭhati  || Vi_86.20 ||
