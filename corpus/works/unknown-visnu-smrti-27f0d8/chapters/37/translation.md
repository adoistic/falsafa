**Vishnu Smriti 37**

Speaking an untruth in self-praise. // Vi_37.1 //

Slander reported to the king. // Vi_37.2 //

And false accusation against a teacher. // Vi_37.3 //

Censuring the Veda. // Vi_37.4 //

And abandoning what has been studied. // Vi_37.5 //

And abandoning the sacred fire, father, mother, son, or wife. // Vi_37.6 //

Eating forbidden food and what ought not to be eaten. // Vi_37.7 //

Stealing another's property. // Vi_37.8 //

Having intercourse with another's wife. // Vi_37.9 //

Sacrificing for one for whom it is forbidden to sacrifice. // Vi_37.10 //

Living by a forbidden occupation. // Vi_37.11 //

And accepting gifts from unworthy persons. // Vi_37.12 //

Killing a Kṣatriya, Vaiśya, Śūdra, or a cow. // Vi_37.13 //

Selling what is forbidden to be sold. // Vi_37.14 //

The supersession of an elder brother by a younger brother who marries first. // Vi_37.15 //

The act of marrying before an elder brother. // Vi_37.16 //

And giving a maiden in marriage to such a one. // Vi_37.17 //

And officiating as a priest for such a one. // Vi_37.18 //

The state of a *vrātya*. // Vi_37.19 //

Teaching for a fee. // Vi_37.20 //

And receiving instruction from a paid teacher. // Vi_37.21 //

Having authority over all mines. // Vi_37.22 //

Operating great machines. // Vi_37.23 //

Harming trees, shrubs, creepers, climbing plants, and herbs. // Vi_37.24 //

Living by one's wife. // Vi_37.25 //

And engaging in black magic and incantations. // Vi_37.26 //

Undertaking rites for one's own sake. // Vi_37.27 //

Not maintaining the sacred fires. // Vi_37.28 //

Not discharging the debts to the gods, sages, and ancestors. // Vi_37.29 //

Studying unrighteous scriptures. // Vi_37.30 //

Atheism. // Vi_37.31 //

Living as a professional actor or singer. // Vi_37.32 //

Consorting with a woman who drinks liquor. // Vi_37.33 //

These are minor sins (*upapātaka*). // Vi_37.34 //

> Those who commit these minor sins shall perform the *cāndrāyaṇa* penance; /
> They shall also perform the *parāka* penance, or they shall perform the Gosava sacrifice. || Vi_37.36 ||
