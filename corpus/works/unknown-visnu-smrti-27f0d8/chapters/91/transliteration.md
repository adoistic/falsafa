Vi_91

atha kūpakartus tatpravṛtte pānīye duṣkṛtasyārdhaṃ vinaśyati // Vi_91.1 //

taḍāgakṛn nityatṛpto vāruṇaṃ lokam aśnute // Vi_91.2 //

jalapradaḥ sadā tṛpto bhavati // Vi_91.3 //

vṛkṣāropayitur vṛkṣāḥ paraloke putrā bhavanti // Vi_91.4 //

vṛkṣaprado vṛkṣaprasūnair devān prīṇayati // Vi_91.5 //

phalaiś cātithīn // Vi_91.6 //

chāyayā cābhyāgatān // Vi_91.7 //

deve varṣaty udakena pitṝn // Vi_91.8 //

setukṛt svargam āpnoti // Vi_91.9 //

devāyatanakārī yasya devasyāyatanaṃ karoti tasyaiva lokam āpnoti // Vi_91.10 //

sudhāsiktaṃ kṛtvā yaśasā virājate // Vi_91.11 //

vicitraṃ kṛtvā gandharvalokam āpnoti // Vi_91.12 //

puṣpapradānena śrīmān bhavati // Vi_91.13 //

anulepanapradānena kīrtimān // Vi_91.14 //

dīpapradānena cakṣuṣmān sarvatrojjvalaś ca // Vi_91.15 //

annapradānena balavān // Vi_91.16 //

devanirmālyāpanayanāt godānaphalam āpnoti // Vi_91.17 //

devagṛhamārjanāt tadupalepanāt brāhmaṇocchiṣṭamārjanāt pādaśaucād akalyaparicaraṇāc ca // Vi_91.18 //

kūpārāmataḍāgeṣu devatāyataneṣu ca  /

punaḥ saṃskārakartā ca labhate maulikaṃ phalam  || Vi_91.19 ||
