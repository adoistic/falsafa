Vi_32

rājartvikśrotriyādharmapratiṣedhyupādhyāyapitṛvyamātāmahamātulaśvaśurajyeṣṭhabhrātṛsaṃbandhinaś cācāryavat // Vi_32.1 //

patnya eteṣāṃ savarṇāḥ // Vi_32.2 //

mātṛṣvasā pitṛṣvasā jyeṣṭhā svasā ca // Vi_32.3 //

śvaśurapitṛvyamātulartvijāṃ kanīyasāṃ pratyutthānam evābhivādanam // Vi_32.4 //

hīnavarṇānāṃ gurupatnīnāṃ dūrād abhivādanaṃ na pādopasparśanam // Vi_32.5 //

gurupatnīnāṃ gotrotsādanāñjanakeśasaṃyamanapādaprakṣālanādīni na kuryāt // Vi_32.6 //

asaṃstutāpi parapatnī bhaginīti vācyā putrīti māteti vā // Vi_32.7 //

na ca gurūṇāṃ tvam iti brūyāt // Vi_32.8 //

tadatikrame nirāhāro divasānte taṃ prasādyāśnīyāt // Vi_32.9 //

na ca guruṇā saha vigṛhya kathāḥ kuryāt // Vi_32.10 //

na caivāsya parīvādam // Vi_32.11 //

na cānabhipretam // Vi_32.12 //

gurupatnī tu yuvatir nābhivādyeha pādayoḥ  /

pūrṇaviṃśativarṣeṇa guṇadoṣau vijānatā  || Vi_32.13 ||

kāmaṃ tu gurupatnīnāṃ yuvatīnāṃ yuvā bhuvi  /

vidhivad vandanaṃ kuryād asāv aham iti bruvan  || Vi_32.14 ||

viproṣya pādagrahaṇam anvahaṃ cābhivādanam  /

gurudāreṣu kurvīta satāṃ dharmam anusmaran  || Vi_32.15 ||

vittaṃ bandhur vayaḥ karma vidyā bhavati pañcamī  /

etāni mānasthānāni garīyo yad yad uttaram  || Vi_32.16 ||

brāhmaṇaṃ daśavarṣaṃ ca śatavarṣaṃ ca bhūmipam  /

pitāputrau vijānīyād brāhmaṇas tu tayoḥ pitā  || Vi_32.17 ||

viprāṇāṃ jñānato jyaiṣṭhyaṃ kṣatriyāṇāṃ tu vīryataḥ  /

vaiśyānāṃ dhānyadhanataḥ śūdrāṇām eva janmataḥ  || Vi_32.18 ||
