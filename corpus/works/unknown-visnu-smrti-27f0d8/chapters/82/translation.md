**Vishnu Smriti 82**

In a rite for the gods, one shall not examine a Brāhmaṇa. // Vi_82.1 //

In a rite for the ancestors, one shall examine him with effort. // Vi_82.2 //

He shall avoid those with deficient or superfluous limbs. // Vi_82.3 //

And those who engage in forbidden occupations. // Vi_82.4 //

Those who follow the cat-vow (hypocrites). // Vi_82.5 //

Those who wear false religious marks. // Vi_82.6 //

Those who live by astrology. // Vi_82.7 //

And temple-priests. // Vi_82.8 //

Physicians. // Vi_82.9 //

Sons of an unmarried woman. // Vi_82.10 //

And their sons. // Vi_82.11 //

Those who sacrifice for many. // Vi_82.12 //

Those who sacrifice for a whole village. // Vi_82.13 //

Those who sacrifice for a Śūdra. // Vi_82.14 //

Those who sacrifice for one for whom it is forbidden to sacrifice. // Vi_82.15 //

*Vrātyas* (those for whom the initiatory rites have not been performed in time). // Vi_82.16 //

And those who sacrifice for them. // Vi_82.17 //

Those who officiate at sacrifices on festival days. // Vi_82.18 //

Informers. // Vi_82.19 //

Paid teachers. // Vi_82.20 //

And those taught by a paid teacher. // Vi_82.21 //

Those nourished by a Śūdra's food. // Vi_82.22 //

Those who associate with fallen persons. // Vi_82.23 //

Those who do not study the Veda. // Vi_82.24 //

Those who have abandoned their twilight worship. // Vi_82.25 //

Servants of the king. // Vi_82.26 //

Naked men. // Vi_82.27 //

Those who quarrel with their father. // Vi_82.28 //

And those who have abandoned their father, mother, preceptor, sacred fire, or personal Vedic study. // Vi_82.29 //

> These are declared to be the dregs of Brāhmaṇas, the *paṅktidūṣakas* (पङ्क्तिदूषकs) (defilers of the row); /
> A wise man shall avoid these with effort in the rite of *śrāddha* (श्राद्ध). || Vi_82.30 ||
