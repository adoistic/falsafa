Vi_8

athāsākṣiṇaḥ // Vi_8.1 //

na rājaśrotriyapravrajitakitavataskaraparādhīnastrībālasāhasikātivṛddhamattonmatābhiśastapatitakṣuttṛṣṇārtavyasanirāgāndhāḥ // Vi_8.2 //

ripumitrārthasaṃbandhivikarmadṛṣṭadoṣasahāyāś ca // Vi_8.3 //

anirdiṣṭas tu sākṣitve yaś copetya brūyāt // Vi_8.4 //

ekaś cāsākṣī // Vi_8.5 //

steyasāhasavāgdaṇḍapāruṣyasaṃgrahaṇeṣu sākṣiṇo na parīkṣyāḥ // Vi_8.6 //

atha sākṣiṇaḥ // Vi_8.7 //

kulajā vṛttavittasaṃpannā yajvānas tapasvinaḥ putriṇo dharmajñā adhīyānāḥ satyavantas traividyavṛddhāś ca // Vi_8.8 //

abhihitaguṇasaṃpanna ubhayānumata eko 'pi // Vi_8.9 //

dvayor vivadamānayor yasya pūrvavādas tasya sākṣiṇaḥ praṣṭavyāḥ // Vi_8.10 //

ādharyaṃ kāryavaśād yatra pūrvapakṣasya bhavet tatra prativādino 'pi // Vi_8.11 //

uddiṣṭasākṣiṇi mṛte deśāntaragate ca tadabhihitaśrotāraḥ pramāṇam // Vi_8.12 //

samakṣadarśanāt sākṣī śravaṇād vā // Vi_8.13 //

sākṣiṇaś ca satyena pūyante // Vi_8.14 //

varṇināṃ yatra vadhas tatrānṛtena // Vi_8.15 //

tatpāvanāya kūśmāṇḍībhir dvijo 'gniṃ ghṛtena juhuyāt // Vi_8.16 //

śūdra ekāhikam godaśakasya grāsaṃ dadyāt // Vi_8.17 //

svabhāvavikṛtau mukhavarṇavināśe 'saṃbaddhapralāpe ca kūṭasākṣiṇaṃ vidyāt // Vi_8.18 //

sākṣiṇaś cāhūya ādityodaye kṛtaśapathān pṛcchet // Vi_8.19 //

brūhīti brāhmaṇaṃ pṛcchet // Vi_8.20 //

satyaṃ brūhīti rājanyam // Vi_8.21 //

gobījakāñcanair vaiśyam // Vi_8.22 //

sarvamahāpātakais tu śūdram // Vi_8.23 //

sākṣiṇaś ca śrāvayet // Vi_8.24 //

ye mahāpātakināṃ lokā ye copapātakināṃ te kūṭasākṣiṇām api // Vi_8.25 //

jananamaraṇāntare kṛtasukṛtahāniś ca // Vi_8.26 //

satyenādityas tapati // Vi_8.27 //

satyena bhāti candramāḥ // Vi_8.28 //

satyena vāti pavanaḥ // Vi_8.29 //

satyena bhūr dhārayati // Vi_8.30 //

satyenāpas tiṣṭhanti // Vi_8.31 //

satyenāgniḥ // Vi_8.32 //

khaṃ ca satyena // Vi_8.33 //

satyena devāḥ // Vi_8.34 //

satyena yajñāḥ // Vi_8.35 //

aśvamedhasahasraṃ ca satyaṃ ca tulayā dhṛtam  /

aśvamedhasahasrād dhi satyam eva viśiṣyate  || Vi_8.36 ||

jānanto 'pi hi ye sākṣye tūṣṇīṃbhūtā udāsate  /

te kūṭasākṣiṇāṃ pāpais tulyā daṇḍena cāpy atha  || Vi_8.37 ||

evaṃ hi sākṣiṇaḥ pṛcched varṇānukramato nṛpaḥ  || Vi_8.38 ||

yasyocuḥ sākṣiṇaḥ satyāṃ pratijñāṃ sa jayī bhavet  /

anyathā vādino yasya dhruvas tasya parājayaḥ  || Vi_8.39 ||

bahutvaṃ pratigṛhṇīyāt sākṣidvaidhe narādhipaḥ  /

sameṣu ca guṇotkṛṣṭān guṇidvaidhe dvijottamān  || Vi_8.40 ||

yasmin yasmin vivāde tu kūṭasākṣy anṛtaṃ vadet  /

tattatkāryaṃ nivarteta kṛtaṃ cāpy akṛtaṃ bhavet  || Vi_8.41 ||
