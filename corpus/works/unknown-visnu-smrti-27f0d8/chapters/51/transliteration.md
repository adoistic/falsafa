Vi_51

surāpaḥ sarvakarmavarjitaḥ kaṇān varṣam aśnīyāt // Vi_51.1 //

malānāṃ madyānāṃ cānyatamasya prāśane cāndrāyaṇaṃ kuryāt // Vi_51.2 //

laśunapalāṇḍugṛñjanaitadgandhiviḍvarāhagrāmakukkuṭavānaragomāṃsabhakṣaṇe ca // Vi_51.3 //

sarveṣv eteṣu dvijānāṃ prāyaścittānte bhūyaḥ saṃskāraṃ kuryāt // Vi_51.4 //

vapanamekhalādaṇḍabhaikṣyacaryāvratāni punaḥsaṃskārakarmaṇi varjanīyāni // Vi_51.5 //

śaśakaśalyakagodhākhaḍgakūrmavarjaṃ pañcanakhamāṃsāśane saptarātram upavaset // Vi_51.6 //

gaṇagaṇikāstenagāyanānnāni bhuktvā saptarātraṃ payasā varteta // Vi_51.7 //

takṣakānnaṃ carmakartuś ca // Vi_51.8 //

vārdhuṣikakadaryadīkṣitabaddhanigaḍābhiśastaṣaṇḍhānāṃ ca // Vi_51.9 //

puṃścalīdāmbhikacikitsakalubdhakakrūrogrocchiṣṭabhojināṃ ca // Vi_51.10 //

avīrastrī suvarṇakārasapatnapatitānāṃ ca // Vi_51.11 //

piśunānṛtavādikṣatadharmātmarasavikrayiṇāṃ ca // Vi_51.12 //

śailūṣatantuvāyakṛtaghnarajakānāṃ ca // Vi_51.13 //

karmakāraniṣādaraṅgāvatārivaiṇaśastravikrayiṇāṃ ca // Vi_51.14 //

śvajīviśauṇḍikatailikacailanirṇejakānāṃ ca // Vi_51.15 //

rajasvalāsahopapativeśmānāṃ ca // Vi_51.16 //

bhrūṇaghnāvekṣitam udakyāsaṃspṛṣṭaṃ patatriṇāvalīḍhaṃ śunā saṃspṛṣṭaṃ gavāghrātaṃ ca // Vi_51.17 //

kāmataḥ padā spṛṣṭam avakṣutam // Vi_51.18 //

mattakruddhāturāṇāṃ ca // Vi_51.19 //

anarcitaṃ vṛthā māṃsaṃ ca // Vi_51.20 //

pāṭhīnarohitarājīvasiṃhatuṇḍaśakulavarjaṃ sarvamatsyamāṃsāśane trirātram upavaset // Vi_51.21 //

sarvajalajamāṃsāśane ca // Vi_51.22 //

āpaḥ surābhāṇḍasthāḥ pītvā saptarātraṃ śaṅkhapuṣpīśṛtaṃ payaḥ pibet // Vi_51.23 //

madyabhāṇḍasthāś ca pañcarātram // Vi_51.24 //

somapaḥ surāpasyāghrāya gandham udakamagnas trir aghamarṣaṇaṃ japtvā ghṛtaprāśanam ācaret // Vi_51.25 //

kharoṣṭrakākamāṃsāśane cāndrāyaṇaṃ kuryāt // Vi_51.26 //

prāśyājñātaṃ sūnāsthaṃ śuṣkamāṃsaṃ ca // Vi_51.27 //

kravyādamṛgapakṣimāṃsāśane taptakṛcchram // Vi_51.28 //

kalaviṅkaplavacakravākahaṃsarajjudālasārasadātyūhaśukasārikābakabalākākokilakhañjarīṭāśane trirātram upavaset // Vi_51.29 //

ekaśaphobhayadantāśane ca // Vi_51.30 //

tittirikapiñjalalāvakavartikāmayūravarjaṃ sarvapakṣimāṃsāśane cāhorātram // Vi_51.31 //

kīṭāśane dinam ekaṃ brahmasuvarcalāṃ pibet // Vi_51.32 //

śunāṃ māṃsāśane ca // Vi_51.33 //

chatrākakavakāśane sāṃtapanam // Vi_51.34 //

yavagodhūmapayovikāraṃ snehāktaṃ śuktaṃ khāṇḍavaṃ ca varjayitvā yat paryuṣitaṃ tat prāśyopavaset // Vi_51.35 //

vraścanāmedhyaprabhavān lohitāṃś ca vṛkṣaniryāsān // Vi_51.36 //

śālūkavṛthākṛsarasaṃyāvapāyasāpūpaśaṣkulīdevānnāni havīṃṣi ca // Vi_51.37 //

goajāmahiṣīvarjaṃ sarvapayāṃsi ca // Vi_51.38 //

anirdaśāhāni tāny api // Vi_51.39 //

syandinīsandhinīvivatsākṣīraṃ ca // Vi_51.40 //

amedhyabhujaś ca // Vi_51.41 //

dadhivarjaṃ kevalāni ca śuktāni // Vi_51.42 //

brahmacaryāśramī śrāddhabhojane trirātram upavaset // Vi_51.43 //

dinam ekaṃ codake vaset // Vi_51.44 //

madhumāṃsāśane prājāpatyam // Vi_51.45 //

biḍālakākanakulākhūcchiṣṭabhakṣaṇe brahmasuvarcalāṃ pibet // Vi_51.46 //

śvocchiṣṭāśane dinam ekam upoṣitaḥ pañcagavyaṃ pibet // Vi_51.47 //

pañcanakhaviṇmūtrāśane saptarātram // Vi_51.48 //

āmaśrāddhāśane trirātraṃ payasā varteta // Vi_51.49 //

brāhmaṇaḥ śūdrocchiṣṭāśane saptarātram // Vi_51.50 //

vaiśyocchiṣṭāśane pañcarātram // Vi_51.51 //

rājanyocchiṣṭāśane trirātram // Vi_51.52 //

brāhmaṇocchiṣṭāśane tv ekāham // Vi_51.53 //

rājanyaḥ śūdrocchiṣṭāśī pañcarātram // Vi_51.54 //

vaiśyocchiṣṭāśī trirātram // Vi_51.55 //

vaiśyaḥ śūdrocchiṣṭāśī ca // Vi_51.56 //

caṇḍālānnaṃ bhuktvā trirātram upavaset // Vi_51.57 //

siddhaṃ bhuktvā parākaḥ // Vi_51.58 //

asaṃskṛtān paśūn mantrair nādyād vipraḥ kathaṃcana  /

mantrais tu saṃskṛtān adyāc chāśvataṃ vidhim āsthitaḥ  || Vi_51.59 ||

yāvanti paśuromāṇi tāvat kṛtveha māraṇam  /

vṛthā paśughnaḥ prāpnoti pretya ceha ca niṣkṛtim  || Vi_51.60 ||

yajñārthaṃ paśavaḥ sṛṣṭāḥ svayam eva svayaṃbhuvā  /

yajño hi bhūtyai sarvasya tasmād yajñe vadho 'vadhaḥ  || Vi_51.61 ||

na tādṛśaṃ bhavaty eno mṛgahantur dhanārthinaḥ  /

yādṛśaṃ bhavati pretya vṛthā māṃsāni khādataḥ  || Vi_51.62 ||

oṣadhyaḥ paśavo vṛkṣās tiryañcaḥ pakṣiṇas tathā  /

yajñārthaṃ nidhanam prāptāḥ prāpnuvanty ucchritīḥ punaḥ  || Vi_51.63 ||

madhuparke ca yajñe ca pitṛdaivatakarmaṇi  /

atraiva paśavo hiṃsyā nānyatreti kathaṃcana  || Vi_51.64 ||

yajñārtheṣu paśūn hiṃsan vedatattvārthavid dvijaḥ  /

ātmānaṃ ca paśūṃś caiva gamayaty uttamāṃ gatim  || Vi_51.65 ||

gṛhe gurāv araṇye vā nivasann ātmavān dvijaḥ  /

nāvedavihitāṃ hiṃsām āpady api samācaret  || Vi_51.66 ||

yā vedavihitā hiṃsā niyatāsmiṃś carācare  /

ahiṃsām eva tāṃ vidyād vedād dharmo hi nirbabhau  || Vi_51.67 ||

yo 'hiṃsakāni bhūtāni hinasty ātmasukhecchayā  /

sa jīvaṃś ca mṛtaś caiva na kvacit sukham edhate  || Vi_51.68 ||

yo bandhanavadhakleśān prāṇināṃ na cikīrṣati  /

sa sarvasya hitaprepsuḥ sukham atyantam aśnute  || Vi_51.69 ||

yad dhyāyati yat kurute ratiṃ badhnāti yatra ca  /

tad evāpnoty ayatnena yo hinasti na kiṃcana  || Vi_51.70 ||

nākṛtvā prāṇināṃ hiṃsāṃ māṃsam utpadyate kvacit  /

na ca prāṇivadhaḥ svargyas tasmān māṃsaṃ vivarjayet  || Vi_51.71 ||

samutpattiṃ ca māṃsasya vadhabandhau ca dehinām  /

prasamīkṣya nivarteta sarvamāṃsasya bhakṣaṇāt  || Vi_51.72 ||

na bhakṣayati yo māṃsaṃ vidhiṃ hitvā piśācavat  /

sa loke priyatāṃ yāti vyādhibhiś ca na pīḍyate  || Vi_51.73 ||

anumantā viśasitā nihantā krayavikrayī  /

saṃskartā copahartā ca khādakaś ceti ghātakāḥ  || Vi_51.74 ||

svamāṃsaṃ paramāṃsena yo vardhayitum icchati  /

anabhyarcya pitṝn devān na tato 'nyo 'sty apuṇyakṛt  || Vi_51.75 ||

varṣe varṣe 'śvamedhena yo yajeta śataṃ samāḥ  /

māṃsāni ca na khāded yas tayoḥ puṇyaphalaṃ samam  || Vi_51.76 ||

phalamūlāśanair divyair munyannānāṃ ca bhojanaiḥ  /

na tat phalam avāpnoti yan māṃsaparivarjanāt  || Vi_51.77 ||

māṃ sa bhakṣayitāmutra yasya māṃsam ihādmy aham  /

etan māṃsasya māṃsatvaṃ pravadanti manīṣiṇaḥ  || Vi_51.78 ||
