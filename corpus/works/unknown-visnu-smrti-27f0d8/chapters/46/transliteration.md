Vi_46

atha kṛcchrāṇi bhavanti // Vi_46.1 //

tryahaṃ nāśnīyāt // Vi_46.2 //

pratyahaṃ ca triṣavaṇaṃ snānam ācaret // Vi_46.3 //

triḥ pratisnānam apsu majjanam // Vi_46.4 //

magnas trir aghamarṣaṇaṃ japet // Vi_46.5 //

divā sthitas tiṣṭhet // Vi_46.6 //

rātrāv āsīnaḥ // Vi_46.7 //

karmaṇo 'nte payasvinīṃ dadyāt // Vi_46.8 //

ity aghmarṣaṇam // Vi_46.9 //

tryahaṃ sāyaṃ tryahaṃ prātas tryaham ayācitam aśnīyāt / eṣa prājāpatyaḥ // Vi_46.10 //

tryaham uṣṇāḥ pibed apas tryaham uṣṇaṃ ghṛtaṃ tryaham uṣṇaṃ payas tryahaṃ ca nāśnīyād eṣa taptakṛcchraḥ // Vi_46.11 //

etair eva śītaiḥ śītakṛcchraḥ // Vi_46.12 //

kṛcchrātikṛcchraḥ payasā divasaikaviṃśatikṣapaṇam // Vi_46.13 //

udakasaktūnāṃ māsābhyavahāreṇodakakṛcchraḥ // Vi_46.14 //

bisābhyavahāreṇa mūlakṛcchraḥ // Vi_46.15 //

bilvābhyavahāreṇa śrīphalakṛcchraḥ // Vi_46.16 //

padmākṣair vā // Vi_46.17 //

nirāhārasya dvādaśāhena parākaḥ // Vi_46.18 //

gomūtraṃ gomayaṃ kṣīraṃ dadhi sarpiḥ kuśodakāny ekadivasam aśnīyāt / dvitīyam upavaset / etat sāṃtapanam // Vi_46.19 //

gomūtrādibhiḥ pratyaham abhyastair mahāsāṃtapanam // Vi_46.20 //

tryahābhyastaiś cātisāṃtapanam // Vi_46.21 //

piṇyākācāmatakrodakasaktūnām upavāsāntarito 'bhyavahāras tulāpuruṣaḥ // Vi_46.22 //

kuśapalāśodumbarapadmaśaṅkhapuṣpīvaṭabrāhmīsuvarcalāpatraiḥ kvathitasyāmbhasaḥ pratyekaṃ pānena parṇakṛcchraḥ // Vi_46.23 //

kṛcchrāṇy etāni sarvāṇi kurvīta kṛtavāpanaḥ  /

nityaṃ triṣavaṇasnāyī adhaḥśāyī jitendriyaḥ  || Vi_46.24 ||

strīśūdrapatitānāṃ ca varjayec cātibhāṣaṇam  /

pavitrāṇi japen nityaṃ juhuyāc caiva śaktitaḥ  || Vi_46.25 ||
