**Vishnu Smriti 77**

The sun's transit into a new sign. // Vi_77.1 //

The two equinoxes. // Vi_77.2 //

Especially the two solstices. // Vi_77.3 //

The *Vyatīpāta* (व्यतीपात) astronomical conjunction. // Vi_77.4 //

One's natal star. // Vi_77.5 //

And a time of prosperity. // Vi_77.6 //

> These times for *śrāddha* (श्राद्ध) Prajāpati declared to be optional (*kāmya*); /
> What is given at a *śrāddha* (श्राद्ध) on these days is destined for eternity. || Vi_77.7 ||
>
> A *śrāddha* (श्राद्ध) should not be performed by the wise at twilight or at night; /
> But it should be performed even at those times, if there is a sighting of Rāhu (an eclipse). || Vi_77.8 ||
>
> For a *śrāddha* (श्राद्ध) given at the sighting of Rāhu lasts as long as the moon and stars; /
> It is meritorious and fulfills all desires for the ancestors. || Vi_77.9 ||
