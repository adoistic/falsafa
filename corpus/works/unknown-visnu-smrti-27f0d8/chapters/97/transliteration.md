Vi_97

ūrusthottānacaraṇaḥ savye kare karam itaraṃ nyasya tālusthācalajihvo dantair dantān asaṃspṛśan svaṃ nāsikāgraṃ paśyan diśaś cānavalokayan vibhīḥ praśāntātmā caturviṃśatyā tattvair vyatītaṃ cintayet // Vi_97.1 //

nityam atīndriyam aguṇaṃ śabdasparśarūparasagandhātītaṃ sarvajñam atisthūlam // Vi_97.2 //

sarvagam atisūkṣmam // Vi_97.3 //

sarvataḥpāṇipādaṃ sarvato 'kṣiśiromukhaṃ sarvataḥ sarvendriyaśaktim // Vi_97.4 //

evaṃ dhyāyet // Vi_97.5 //

dhyānaniratasya ca saṃvatsareṇa yogāvirbhāvo bhavati // Vi_97.6 //

atha nirākāre lakṣabandhaṃ kartuṃ na śaknoti, tadā pṛthivyaptejovāyvākāśamanobuddhyātmāvyaktapuruṣāṇāṃ pūrvaṃ pūrvaṃ dhyātvā tatra labdhalakṣaḥ tatparityajyāparam aparaṃ dhyāyet // Vi_97.7 //

evaṃ puruṣadhyānam ārabheta // Vi_97.8 //

tatrāpy asamarthaḥ svahṛdayapadmasya avāṅmukhasya madhye dīpavat puruṣaṃ dhyāyet // Vi_97.9 //

tatrāpy asamartho bhagavantaṃ vāsudevaṃ kirīṭinaṃ kuṇḍalinam aṅgadinaṃ śrīvatsāṅkaṃ vanamālāvibhūṣitoraskaṃ saumyarūpaṃ caturbhujaṃ śaṅkhacakragadāpadmadharaṃ caraṇamadhyagatabhuvaṃ dhyāyet // Vi_97.10 //

yad dhyāyati tad āpnotīti dhyānaguhyam // Vi_97.11 //

tasmāt sarvam eva kṣaraṃ tyaktvā akṣaram eva dhyāyet // Vi_97.12 //

na ca puruṣaṃ vinā kiṃcid apy akṣaram asti // Vi_97.13 //

taṃ prāpya mukto bhavati // Vi_97.14 //

puram ākramya sakalaṃ śete yasmān mahāprabhuḥ  /

tasmāt puruṣa ity evaṃ procyate tattvacintakaiḥ  || Vi_97.15 ||

prāgrātrāpararātreṣu yogī nityam atandritaḥ  /

dhyāyeta puruṣaṃ viṣṇuṃ nirguṇaṃ pañcaviṃśakam  || Vi_97.16 ||

tattvātmānam agamyaṃ ca sarvatattvavivarjitam  /

aśaktaṃ sarvabhṛc caiva nirguṇaṃ guṇabhoktṛ ca  || Vi_97.17 ||

bahir antaś ca bhūtānām acaraṃ caram eva ca  /

sūkṣmatvāt tadavijñeyaṃ dūrasthaṃ cāntike ca tat  || Vi_97.18 ||

avibhaktaṃ ca bhūtena vibhaktam iva ca sthitam  /

bhūtabhavyabhavadrūpaṃ grasiṣṇu prabhaviṣṇu ca  || Vi_97.19 ||

jyotiṣām api tajjyotis tamasaḥ param ucyate  /

jñānaṃ jñeyaṃ jñānagamyaṃ hṛdi sarvasya dhiṣṭhitam  || Vi_97.20 ||

iti kṣetraṃ tathā jñānaṃ jñeyaṃ coktaṃ samāsataḥ  /

madbhakta etadvijñāya madbhāvāyopapadyate  || Vi_97.21 ||
