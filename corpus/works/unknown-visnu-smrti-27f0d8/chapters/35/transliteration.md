brahmahatyā surāpānaṃ brāhmaṇasuvarṇaharaṇaṃ gurudāragamanam iti mahāpātakāni // Vi_35.1 //

tatsaṃyogaś ca // Vi_35.2 //

saṃvatsareṇa patati patitena sahācaran // Vi_35.3 //

ekayānabhojanāśanaśayanaiḥ // Vi_35.4 //

yaunasrauvamukhyaiḥ saṃbandhais tu sadya eva // Vi_35.5 //

aśvamedhena śuddhyeyur mahāpātakinas tv ime  /

pṛthivyāṃ sarvatīrthānāṃ tathānusaraṇena ca  || Vi_35.6 ||
