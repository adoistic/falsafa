Vi_6

athottamarṇo 'dhamarṇād yathādattam arthaṃ gṛhṇīyāt // Vi_6.1 //

dvikaṃ trikaṃ catuṣkaṃ pañcakaṃ ca śataṃ varṇānukrameṇa pratimāsam // Vi_6.2 //

sarve varṇā vā svapratipannāṃ vṛddhiṃ dadyuḥ // Vi_6.3 //

akṛtām api vatsarātikrameṇa yathāvihitam // Vi_6.4 //

ādhyupabhoge vṛddhyabhāvaḥ // Vi_6.5 //

daivarājopaghātād ṛte vinaṣṭam ādhim uttamarṇo dadyāt // Vi_6.6 //

antavṛddhau praviṣṭāyām api // Vi_6.7 //

na sthāvaram ādhim ṛte vacanāt // Vi_6.8 //

gṛhītadhanapraveśārtham eva yat sthāvaraṃ dattaṃ tat gṛhītadhanapraveśe dadyāt // Vi_6.9 //

dīyamānaṃ prayuktam artham uttamarṇasyāgṛhṇatas tataḥ paraṃ na vardhate // Vi_6.10 //

hiraṇyasya parā vṛddhir dviguṇā // Vi_6.11 //

dhānyasya triguṇā // Vi_6.12 //

vastrasya caturguṇā // Vi_6.13 //

rasasyāṣṭaguṇā // Vi_6.14 //

saṃtatiḥ strīpaśūnām // Vi_6.15 //

kiṇvakārpāsasūtracarmāyudheṣṭakāṅgarāṇām akṣayā // Vi_6.16 //

anuktānāṃ dviguṇā // Vi_6.17 //

prayuktam arthaṃ yathā kathaṃcit sādhayan na rājño vācyaḥ syāt // Vi_6.18 //

sādhyamānaś ced rājānam abhigacchet tatsamaṃ daṇḍyaḥ // Vi_6.19 //

uttamarṇaś ced rājānam iyāt, tadvibhāvito 'dhamarṇo rājñe dhanadaśabhāgasaṃmitaṃ daṇḍaṃ dadyāt // Vi_6.20 //

prāptārthaś cottamarṇo viṃśatitamam aṃśam // Vi_6.21 //

sarvāpalāpy ekadeśavibhāvito 'pi sarvaṃ dadyāt // Vi_6.22 //

tasya ca bhāvanās tisro bhavanti likhitaṃ sākṣiṇaḥ samayakriyā ca // Vi_6.23 //

sasākṣikam āptaṃ sasākṣikam eva dadyāt // Vi_6.24 //

likhtārthe praviṣṭe likhitaṃ pāṭayet // Vi_6.25 //

asamagradāne lekhyāsaṃnidhāne cottamarṇaḥ svalikhitaṃ dadyāt // Vi_6.26 //

dhanagrāhiṇi prete prevrajite dvidaśāḥ samāḥ pravasite vā tatputrapautrair dhanaṃ deyam // Vi_6.27 //

nātaḥ param anicchubhiḥ // Vi_6.28 //

saputrasya vāpy aputrasya vā rikthagrāhī ṛṇaṃ dadyāt // Vi_6.29 //

nirdhanasya strīgrāhī // Vi_6.30 //

na strī patiputrakṛtam // Vi_6.31 //

na strīkṛtaṃ patiputrau // Vi_6.32 //

na pitā putrakṛtam // Vi_6.33 //

avibhaktaiḥ kṛtam ṛṇaṃ yas tiṣṭhet sa dadyāt // Vi_6.34 //

paitṛkam ṛṇam avibhaktānāṃ bhrātṝṇāṃ ca // Vi_6.35 //

vibhaktāś ca dāyānurūpam aṃśam // Vi_6.36 //

gopaśauṇḍikaśailūṣarajakavyādhastrīṇāṃ patir dadyāt // Vi_6.37 //

vākpratipannaṃ nādeyaṃ kasyacit // Vi_6.38 //

kuṭumbārthe kṛtaṃ ca // Vi_6.39 //

yo gṛhītvā ṛṇaṃ sarvaṃ śvo dāsyāmīti sāmakam  /

na dadyāl lobhataḥ paścāt tathā vṛddhim avāpnuyāt  || Vi_6.40 ||
