Vi_49

mārgaśīrṣaśuklaikādaśyām upoṣito dvādaśyāṃ bhagavantaṃ śrīvāsudevam arcayet // Vi_49.1 //

puṣpadhūpānulepanadīpanaivedyaiḥ vahnibrāhmaṇatarpaṇaiś ca // Vi_49.2 //

vratam etat saṃvatsaraṃ kṛtvā pāpebhyaḥ pūto bhavati // Vi_49.3 //

yāvaj jīvaṃ kṛtvā śvetadvīpam āpnoti // Vi_49.4 //

ubhayapakṣadvādaśīṣv evaṃ saṃvatsareṇa svargalokam āpnoti // Vi_49.5 //

yāvaj jīvaṃ kṛtvā viṣṇulokam // Vi_49.6 //

evam eva pañcadaśīṣv api // Vi_49.7 //

brahmabhūtam amāvāsyāṃ paurṇamāsyāṃ tathaiva ca  /

yogabhūtaṃ paricaran keśavaṃ mahad āpnuyāt  || Vi_49.8 ||

dṛśyete sahitau yasyāṃ divi candrabṛhaspatī  /

paurṇamāsī tu mahatī proktā saṃvatsare tu sā  || Vi_49.9 ||

tasyāṃ dānopavāsādyam akṣayaṃ parikīrtitam  /

tathaiva dvādaśī śuklā yā syāc chravaṇasaṃyutā  || Vi_49.10 ||
