Vi_15

atha dvādaśa putrā bhavanti // Vi_15.1 //

svakṣetre saṃskṛtāyām utpāditaḥ svayam aurasaḥ prathamaḥ // Vi_15.2 //

niyuktāyāṃ sapiṇḍenottamavarṇena votpāditaḥ kṣetrajo dvitīyaḥ // Vi_15.3 //

putrīkāputras tṛtīyaḥ // Vi_15.4 //

yas tv asyāḥ putraḥ sa me putro bhaved iti yā pitrā dattā sā putrikā // Vi_15.5 //

putrikāvidhiṃ vināpi pratipāditā bhrātṛvihīnā putrikaiva // Vi_15.6 //

paunarbhavaś caturthaḥ // Vi_15.7 //

akṣatā bhūyaḥ saṃskṛtā punarbhūḥ // Vi_15.8 //

bhūyas tv asaṃskṛtāpi parapūrvā // Vi_15.9 //

kānīnaḥ pañcamaḥ // Vi_15.10 //

pitṛgṛhe asaṃskṛtayaivotpāditaḥ // Vi_15.11 //

sa ca pāṇigrāhasya // Vi_15.12 //

gṛhe ca gūḍhotpannaḥ ṣaṣṭhaḥ // Vi_15.13 //

yasya talpajas tasyāsau // Vi_15.14 //

sahoḍhaḥ saptamaḥ // Vi_15.15 //

yā garbhiṇī saṃskriyate tasyāḥ putraḥ // Vi_15.16 //

sa ca pāṇigrāhasya // Vi_15.17 //

dattakaś cāṣṭamaḥ // Vi_15.18 //

sa ca mātāpitṛbhyāṃ yasya dattaḥ // Vi_15.19 //

krītaś ca navamaḥ // Vi_15.20 //

sa ca yena krītaḥ // Vi_15.21 //

svayam upagato daśamaḥ // Vi_15.22 //

sa ca yasyopagataḥ // Vi_15.23 //

apaviddhas tv ekādaśaḥ // Vi_15.24 //

pitrā mātrā ca parityaktaḥ // Vi_15.25 //

sa ca yena gṛhītaḥ // Vi_15.26 //

yatra kvacanotpāditaś ca dvādaśaḥ // Vi_15.27 //

eteṣāṃ pūrvaḥ pūrvaḥ śreyān // Vi_15.28 //

sa eva dāyaharaḥ // Vi_15.29 //

sa cānyān bibhṛyāt // Vi_15.30 //

anūḍhānāṃ svavittānurūpeṇa saṃskāraṃ kuryāt // Vi_15.31 //

patitaklībācikitsyarogavikalās tv abhāgahāriṇaḥ // Vi_15.32 //

rikthagrāhibhis te bhartavyāḥ // Vi_15.33 //

teṣāṃ caurasāḥ putrā bhāgahāriṇaḥ // Vi_15.34 //

na tu patitasya // Vi_15.35 //

patanīye karmaṇi kṛte tv anantarotpannāḥ // Vi_15.36 //

pratilomāsu strīṣu cotpannāś cābhāginaḥ // Vi_15.37 //

tatputrāḥ paitāmahe 'py arthe // Vi_15.38 //

aṃśagrāhibhis te bharaṇīyāḥ // Vi_15.39 //

yaś cārthaharaḥ sa piṇḍadāyī // Vi_15.40 //

ekoḍhānām apy ekasyāḥ putraḥ sarvāsāṃ putra eva // Vi_15.41 //

bhrātṝṇām ekajātānāṃ ca // Vi_15.42 //

putraḥ pitṛvittālābhe 'pi piṇḍaṃ dadyāt // Vi_15.43 //

punnāmno narakād yasmāt pitaraṃ trāyate sutaḥ  /

tasmāt puttra iti proktaḥ svayam eva svayaṃbhuvā  || Vi_15.44 ||

ṛṇam asmin saṃnayaty amṛtatvaṃ ca gacchati  /

pitā putrasya jātasya paśyec cej jīvato mukham  || Vi_15.45 ||

putreṇa lokān jayati pautreṇānantyam aśnute  /

atha putrasya pautreṇa bradhnasyāpnoti viṣṭapam  || Vi_15.46 ||

pautradauhitrayor loke viśeṣo nopapadyate  /

dauhitro 'pi hy aputraṃ taṃ saṃtārayati pautravat  || Vi_15.47 ||
