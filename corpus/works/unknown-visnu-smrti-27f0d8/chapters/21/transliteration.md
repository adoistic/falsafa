Vi_21

athāśaucavyapagame susnātaḥ suprakṣālitapāṇipādaḥ svācāntas tv evaṃvidhān eva brāhmaṇān yathāśakti udaṅmukhān gandhamālyavastrālaṃkārādibhiḥ pūjitān bhojayet // Vi_21.1 //

ekavanmantrān ūhed ekoddiṣṭe // Vi_21.2 //

ucchiṣṭasaṃnidhāv ekam eva tannāmagotrābhyāṃ piṇḍaṃ nirvapet // Vi_21.3 //

bhuktavatsu brāhmaṇeṣu dakṣiṇayābhipūjiteṣu pretanāmagotrābhyāṃ dattākṣayyodakaḥ caturaṅgulapṛthvīḥ tāvadantarāḥ tāvadadhaḥkhātāḥ vitastyāyatāḥ tisraḥ karṣūḥ kuryāt // Vi_21.4 //

karṣūsamīpe cāgnitrayam upasamādhāya paristīrya tatraikaikasmin āhutitrayaṃ juhuyāt // Vi_21.5 //

somāya pitṛmate svadhā namaḥ // Vi_21.6 //

agnaye kavyavāhanāya svathā namaḥ // Vi_21.7 //

yamāyāṅgirase svadhā namaḥ // Vi_21.8 //

sthānatraye ca prāgvat piṇḍanirvapaṇaṃ kuryāt // Vi_21.9 //

annadadhighṛtamadhumāṃsaiḥ karṣūtrayaṃ pūrayitvā etat ta iti japet // Vi_21.10 //

evaṃ mṛtāhe pratimāsaṃ kuryāt // Vi_21.11 //

saṃvatsarānte pretāya tatpitre tatpitāmahāya tatprapitāmahāya ca brāhmaṇān devapūrvān bhojayet // Vi_21.12 //

atrāgnaukaraṇam āvāhanaṃ pādyaṃ ca kuryāt // Vi_21.13 //

saṃsṛjatu tvā pṛthivī samānī ca iti ca pretapādyapātre pitṛpādyapātratraye yojayet // Vi_21.14 //

ucchiṣṭasaṃnidhau piṇḍacatuṣṭayaṃ kuryāt // Vi_21.15 //

brāhmaṇāṃś ca svācāntān dattadakṣiṇāṃś cānuvrajya visarjayet // Vi_21.16 //

tataḥ pretapiṇḍaṃ pādyapātrodakavat piṇḍatraye nidadhyāt // Vi_21.17 //

karṣūtrayasaṃnikrṣe 'py evam eva // Vi_21.18 //

sapiṇḍīkaraṇaṃ māsikārthavat dvādaśāhaṃ śrāddhaṃ kṛtvā trayodaśe 'hni vā kuryāt // Vi_21.19 //

mantravarjaṃ hi śūdrāṇāṃ dvādaśe 'hni // Vi_21.20 //

saṃvatsarābhyantare yady adhimāso bhavet, tadā māsikārthe dinam ekaṃ vardhayet // Vi_21.21 //

sapiṇḍīkaraṇaṃ strīṇāṃ kāryam evaṃ tathā bhavet  /

yāvaj jīvaṃ tathā kuryāc chrāddhaṃ tu prativatsaram  || Vi_21.22 ||

arvāk sapiṇḍīkaraṇaṃ yasya saṃvatsarāt kṛtaṃ  /

tasyāpy annaṃ sodakumbhaṃ dadyād varṣaṃ dvijanmane  || Vi_21.23 ||
