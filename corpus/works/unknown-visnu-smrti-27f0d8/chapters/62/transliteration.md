Vi_62

atha dvijātīnāṃ kanīnikāmūle prājāpatyaṃ nāma tīrtham // Vi_62.1 //

aṅguṣṭhamūle brāhmam // Vi_62.2 //

aṅgulyagre daivam // Vi_62.3 //

tarjanīmūle pitryam // Vi_62.4 //

anagnyuṣṇābhir aphenilābhiḥ aśūdraikakarāvarjitābhir akṣārābhir adbhiḥ śucau deśe svāsīno 'ntarjānu prāṅmukhaś codaṅmukho vā tanmanāḥ sumanāś cācāmet // Vi_62.5 //

brāhmeṇa tīrthena trir ācāmet // Vi_62.6 //

dviḥ pramṛjyāt // Vi_62.7 //

khāny adbhir mūrdhānaṃ hṛdayaṃ spṛśet // Vi_62.8 //

hṛtkaṇṭhatālugābhiś ca yathāsaṃkhyaṃ dvijātayaḥ  /

śudhyeran strī ca śūdraś ca sakṛt spṛṣṭābhir antataḥ  || Vi_62.9 ||
