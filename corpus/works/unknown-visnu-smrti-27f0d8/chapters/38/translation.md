**Vishnu Smriti 38**

Causing injury to a Brāhmaṇa. // Vi_38.1 //

Smelling what ought not to be smelled and liquor. // Vi_38.2 //

Dishonesty. // Vi_38.3 //

Having intercourse with animals. // Vi_38.4 //

And with a man. // Vi_38.5 //

These are sins causing loss of caste (*jātibhraṃśakara*). // Vi_38.6 //

> Having intentionally committed any act causing loss of caste, /
> One shall perform the *sāṃtapana kṛcchra*; unintentionally, the *prājāpatya*. || Vi_38.7 ||
