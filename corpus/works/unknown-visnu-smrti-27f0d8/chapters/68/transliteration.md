Vi_68

candrārkoparāge nāśnīyāt // Vi_68.1 //

snātvā muktayor aśnīyāt // Vi_68.2 //

amuktayor astaṃ gatayoḥ snātvā dṛṣṭvā cāpare 'hni // Vi_68.3 //

na gobrāhmaṇoparāge 'śnīyāt // Vi_68.4 //

na rājño vyasane // Vi_68.5 //

pravasitāgnihotrī yadāgnihotraṃ kṛtaṃ manyate tadāśnīyāt // Vi_68.6 //

yadā kṛtaṃ manyeta vaiśvadevam api // Vi_68.7 //

parvaṇi ca yadā kṛtaṃ manyeta parva // Vi_68.8 //

nāśnīyāc cājīrṇe // Vi_68.9 //

nārdharātre // Vi_68.10 //

na madyāhne // Vi_68.11 //

na saṃdhyayoḥ // Vi_68.12 //

nārdravāsāḥ // Vi_68.13 //

naikavāsāḥ // Vi_68.14 //

na nagnaḥ // Vi_68.15 //

nodake // Vi_68.16 //

notkuṭakaḥ // Vi_68.17 //

na bhinnāsanagataḥ // Vi_68.18 //

na ca śayanagataḥ // Vi_68.19 //

na bhinnabhojane // Vi_68.20 //

notsaṅge // Vi_68.21 //

na bhuvi // Vi_68.22 //

nānārdrakaramukhaś ca // Vi_68.35 //

piben nāñjalinā toyaṃ nātisauhityam ācaret  || Vi_68.47 ||

na tṛtīyam athāśnīta na cāpathyaṃ kathaṃcana  /
