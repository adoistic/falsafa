**Vishnu Smriti 68**

He shall not eat during an eclipse of the moon or sun. // Vi_68.1 //

Having bathed, he shall eat after they are released. // Vi_68.2 //

If they have set while unreleased, having bathed, he shall eat after seeing them on the next day. // Vi_68.3 //

He shall not eat during an eclipse affecting cows or Brāhmaṇas. // Vi_68.4 //

Nor when the king is in distress. // Vi_68.5 //

When a householder who maintains the sacred fire is away from home, he shall eat when he thinks the Agnihotra has been performed. // Vi_68.6 //

And when he thinks the Vaiśvadeva has also been performed. // Vi_68.7 //

And on a festival day, when he thinks the festival rite has been performed. // Vi_68.8 //

He shall not eat when he has indigestion. // Vi_68.9 //

Not at midnight. // Vi_68.10 //

Not at midday. // Vi_68.11 //

Not at the twilights. // Vi_68.12 //

Not with wet clothes. // Vi_68.13 //

Not with a single garment. // Vi_68.14 //

Not naked. // Vi_68.15 //

Not in water. // Vi_68.16 //

Not *squatting*. // Vi_68.17 //

Not seated on a broken seat. // Vi_68.18 //

And not lying on a bed. // Vi_68.19 //

Not in a broken vessel. // Vi_68.20 //

Not on his lap. // Vi_68.21 //

Not on the ground. // Vi_68.22 //

Not in his hand. // Vi_68.23 //

And he shall not eat that to which salt is added directly. // Vi_68.24 //

He shall not scold children. // Vi_68.25 //

Not dainties alone. // Vi_68.26 //

Not food from which the oil has been extracted. // Vi_68.27 //

Not parched grain during the day. // Vi_68.28 //

Not anything connected with sesame at night. // Vi_68.29 //

Not barley-meal with curd. // Vi_68.30 //

Not the vegetable of Kovidāra, Vaṭa, Pippala, or Śāṇa. // Vi_68.31 //

Not without having given. // Vi_68.32 //

Not without having offered. // Vi_68.33 //

Not without having wet feet. // Vi_68.34 //

And not without having wet hands and mouth. // Vi_68.35 //

And while in a state of impurity, he shall not take clarified butter. // Vi_68.36 //

He shall not gaze at the moon, sun, or stars. // Vi_68.37 //

He shall not touch his head. // Vi_68.38 //

He shall not recite the Veda. // Vi_68.39 //

He shall eat facing east. // Vi_68.40 //

Or facing south. // Vi_68.41 //

Having honored the food. // Vi_68.42 //

With a pleasant mind, wearing a garland and anointed. // Vi_68.43 //

He shall not be one who leaves no remainder. // Vi_68.44 //

Except for curd, honey, clarified butter, milk, barley-meal, meat, and sweetmeats. // Vi_68.45 //

> He shall not eat with his wife, nor in the open air, nor standing up; /
> Not while many are watching, nor many from a single vessel. || Vi_68.46 ||
>
> In a deserted house, in a fire-house, in a temple, in any way, /
> He shall not drink water with his cupped hands, nor shall he eat to great excess. || Vi_68.47 ||
>
> He shall not eat a third time, nor anything unwholesome in any way; /
> Not too early, not too late; not in the evening if he has eaten in the morning. || Vi_68.48 ||
>
> He shall not eat food that is spoiled in its nature, nor in a vessel that is spoiled in its nature; /
> Lying down, with his feet raised, or with one leg crossed over the other. || Vi_68.49 ||
