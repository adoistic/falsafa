Vi_89

māsaḥ kārttiko 'gnidaivatyaḥ // Vi_89.1 //

agniś ca sarvadevānāṃ mukham // Vi_89.2 //

tasmāt tu kārttikaṃ māsaṃ bahiḥsnāyī gāyatrījapanirataḥ sakṛd eva haviṣyāśī saṃvatsarakṛtāt pāpāt pūto bhavati // Vi_89.3 //

kārttikaṃ sakalaṃ māsaṃ nityasnāyī jitendriyaḥ  /

japan haviṣyabhugdāntaḥ sarvapāpaiḥ pramucyate  || Vi_89.4 ||
